1. Planes y antecedentes

1.1 Introducción

Actualmente, la aplicación TIS 2000 es una importante fuente de información en los talleres de los Servicios Oficiales Opel.  Las actualizaciones periódicas en DVD proporcionan documentación técnica, así como software para los dispositivos de prueba y las ECU (unidades de control electrónicas).

Se está revisando TIS 2000 para convertirlo en una nueva aplicación disponible a través de Internet.  Las ventajas son claras: información actualizada y evitación de problemas de instalación.  Un paso intermedio en el tránsito a Internet es tener la nueva aplicación, pero aún en DVD y diseñada para una instalación local.  De este modo se reduce la dependencia respecto a la disponibilidad de Internet.

Además, la anterior aplicación TIS 2000 se ha cambiado para compactar las categorías de la información y facilitar un acceso más rápido a documentos específicos.  Ahora encontrará una tabla de contenidos armonizada donde verá no sólo las instrucciones de reparación y desmontaje, sino también las ayudas técnicas, instrucciones de verificación y diagramas de cableado.

2. Instalación (servidor o equipo individual)

Como en TIS 2000, el DVD tiene que instalarse en el servidor o en un ordenador local (instalación individual).  Sin embargo, hay algunas diferencias en el manejo que se describen a continuación.  Otra diferencia fundamental es la necesidad de un registro.  Eso significa que GlobalTIS sólo se puede usar después de que el usuario confirme satisfactoriamente que es un usuario autorizado.  Para más detalles, véase el párrafo 2.1.4.

2.1  Antes de la instalación:

  • El usuario que desee efectuar una instalación o una actualización posterior, debe tener derechos de administración en ese ordenador.
  • Antes de iniciar la instalación, asegúrese de que su sistema Windows esté actualizado con el Service Pack más reciente.
  • El directorio de instalación para GlobalTIS debe estar vacío o no existir.  El nombre del directorio no debe incluir caracteres especiales.
  • Si tiene una llave de hardware (en adelante 'HWK') tipo USB conectada al ordenador, tiene que desconectarla durante la instalación.  En caso contrario, Windows instalará automáticamente software de controladores que afectan al funcionamiento posterior.  Entonces se instalará un controlador incorrecto.

2.2 Durante la instalación:

  • El sistema le preguntará por el idioma. Esta selección sólo afecta al procedimiento de instalación. El idioma de la interfaz de usuario y de los documentos se define de forma diferente.
  • El sistema le mostrará las entradas de los puertos, p. ej., el 'Puerto del servidor Transbase'. No debe cambiarlos salvo que desee modificarlos a propósito.
  • El sistema le mostrará la entrada con el nombre de host.Se debería dejar 'localhost' en un ordenador individual o cambiarse al nombre del equipo en un servidor (vaya a 'Mi PC' en el escritorio, haga clic con el botón derecho del ratón, seleccione el nombre del equipo, pulse el botón 'Cambiar' para obtener el nombre correcto del ordenador).
  • El sistema le mostrará la entrada 'Servidor de correo'.  Esta función se usa para enviar comentarios directamente desde este equipo sin tener que imprimir documentos para un fax.  De forma predeterminada, no es necesario cambiar estos valores.

2.3  Después de la instalación, pero antes del primer inicio de la aplicación:

Una vez terminada la instalación, el usuario debe iniciar la aplicación manualmente desde 'Inicio' - 'Programas' - 'GlobalTIS' - 'GlobalTIS'.

3. Registro:

Sólo los usuarios autorizados pueden trabajar con GlobalTIS.  La autorización se obtiene mediante un proceso de registro.

En el pasdo, un usuario podía trabajar con GlobalTIS después de la conversión de su llave de hardware (HWK). Su número de serie se convertía automáticamente en la autorización (clave de licencia). Por motivos de seguridad, este procedimiento de autorización ya no es viable y debe renovarse lo antes posible.

  • Registro online
  • Registro manual

Cada uno de los campos de entrada marcados con un asterisco (*) deben completarse con los datos requeridos.  Al final del formulario, el usuario puede elegir entre realizar un registro online o un registro manual.  En caso de registro online , los datos relacionados con el usuario se transfieren electrónicamente a una ubicación central, donde se genera una clave de licencia y ésta se devuelve al sistema.  La clave de licencia es un número codificado y tiene que introducirse en el campo para 'clave de licencia'.

En caso de registro manual, el sistema genera automáticamente un documento en formato PDF.  Se puede enviar a la dirección indicada como adjunto a un mensaje de correo electrónico, pero también por fax.  No obstante, es preferible usar el correo electrónico porque su procesamiento es mucho más rápido.

3.1  Suscripciones para una marca

Durante el proceso de registro, el usuario obtendrá una suscripción sólo para 1 marca.  Para cada marca adicional es necesario ampliar el registro, repitiendo de nuevo el proceso de registro.  Una vez ampliada la suscripción, es necesario cerrar GlobalTIS y volver a iniciarlo para que aparezcan los nuevos iconos de la aplicación.

3.2  Cambios en la suscripción

Por eso, si se requieren suscripciones para marcas adicionales, debe modificarse el registro existente.  Para ello, inicie GlobalTIS y seleccione la ficha 'Admin'.  Puede seleccionar una nueva suscripción y pulsar el botón para el registro online o manual.

El mismo procedimiento es necesario si no dispone de suficientes licencias de usuarios autorizados y hay que aumentar el número de usuarios.  Esta cifra limita el número de conexiones simultáneas de usuarios a un servidor.  En la mayoría de los casos, no es necesario modificar este número.

4. Actualización de GlobalTIS

Cuando se publica una nueva versión de GlobalTIS en DVD, representa una actualización de una instalación existente.  Introduzca el DVD en la unidad de DVD.  El procedimiento de actualización se iniciará automáticamente si está activada la opción 'Reproducción automática' en el ordenador.  Si esta función no está activada en su sistema, debe iniciar el ejecutable setup.exe manualmente desde el directorio raíz del DVD de GlobalTIS.  GlobalTIS no busca automáticamente actualizaciones en la unidad de DVD.

5. Inicio de la aplicación

La aplicación puede iniciarse haciendo clic en la entrada correspondiente del menú 'Inicio'.  La entrada predeterminada se encuentra en 'Inicio' - 'Programas' - 'GlobalTIS' - 'GlobalTIS'

5.1  Iniciar un equipo cliente

Si la aplicación se va a iniciar desde un equipo cliente, sólo necesita el programa 'Internet Explorer'.  En el campo de dirección hay que introducir el nombre del servidor, p. ej.: 'http://server-name:port/tis2web'.

Ejemplo: http://asterix:9080/tis2web

El 'puerto' será normalmente '9080' o el valor introducido durante la instalación, al definir el puerto para 'Tomcat'.
La dirección debería guardarse como marcador de página en los favoritos de su navegador.  GlobalTIS sólo debería utilizarse con Microsoft Internet Explorer.

5.2 Iniciar un servidor

Una vez iniciado un servidor tiene que iniciar sesión con un nombre de usuario.  Después del primer inicio tiene que introducir un nuevo usuario de su elección.  Cuando reinicie, puede seleccionar dicho usuario o introducir uno nuevo.

5.3  Entorno Java (Java Runtime Environment)

Es necesario para los módulos

  • Descarga de software
  • Sistema de programación de servicio (SPS)
  • Acceso de seguridad
  • Instantánea
  • Tech2 View
  • Imprimir el Techline

Si ya hay instalada en el equipo una versión de Java superior a la 1.4x, entonces se descargarán automáticamente las aplicaciones Java cuando se inicien los módulos de GlobalTIS.  Esto ocurre sólo una vez o cuando haya una versión revisada disponible en el servidor.
Si aún no hay entorno Java instalado, o la versión es más antigua, será necesario instalarlo primero.  La versión Java se puede descargar desde el menú de 'Ayuda' y la sección: Información general -  Descargar las aplicaciones

5.4  Configuración proxy de Java

Si las aplicaciones Java no se inician, puede ser necesario cambiar la configuración de la aplicación 'Java Web Start'. Es necesario desactivar la configuración del servidor proxy.
'Archivo' - 'Preferencias' - 'General'. Seleccione 'Ninguno' para Proxies.

5.5  Configuración de pantalla

Para favorecer un manejo eficiente y sencillo del software, es primordial una configuración óptima de la pantalla.  Básicamente debe prestar atención a 2 secciones.

Ajuste un tamaño apropiado para el valor del área de documentos.  Seleccione el icono 'Configuración' y cambie el parámetro 'Tamaño en pantalla del documento' hasta que el documento ocupe toda la página.

Cambie el modo de visualización de Internet Explorer a 'Pantalla completa' pulsando la tecla de función 'F11'.

El botón de 'alternar pantalla' le permite cambiar entre el modo de visualización con la tabla de contenidos junto al documento o ver sólo el contenido del documento.  Esto contribuirá al uso óptimo del espacio disponible en la pantalla.  El botón de alternar está disponible en la barra de iconos, como se muestra a continuación:

5.6  Configuración del idioma

El idioma de GlobalTIS está determinado por la configuración del navegador.  Para cambiar el idioma, abra el menú 'Herramientas', seleccione 'Opciones de Internet' y haga clic en el botón 'Idiomas...'.  Siga las instrucciones del navegador.
Para activar el cambio, se necesario que el usuario vuelva a iniciar sesión en la aplicación.

5.7  Configuración del navegador

La versión actual está configurada para una resolución de pantalla de al menos 1024 x 768 píxeles y tamaño de texto 'pequeño' (en Internet Explorer, menú 'Ver' - 'Tamaño de texto').  Si utiliza una configuración distinta, es posible que aparezca una barra de desplazamiento en el borde de la pantalla. Utilice dicha barra para ver el documento completo.

5.8  Administración de sesiones

En TIS 2000 hay una función denominada 'Administración de sesiones'.  Permite a varios usuarios trabajar con TIS 2000 en un entorno independiente.  Se conservan todos los ajustes, como tipo de vehículo, año modelo y criterios de búsqueda, y los otros usuarios pueden recuperar su información sin interferir.

GlobalTIS no ofrece la misma técnica pero sí un mecanismo similar.  Si otro usuario necesita trabajar con el sistema pero no quiere volver a configurar todos los ajustes definidos previamente, sólo tiene que abrir otra ventana en Internet Explorer e iniciar sesión con su ID personal.


5.9  Funcionalidad de los botones Atrás y Adelante

Los botones Atrás y Adelante de Internet Explorer no funcionan durante la ejecución de GlobalTIS.  Por lo tanto, no deben utilizarse.  En su lugar, utilice el símbolo correspondiente en la barra de iconos.

 

5.10  Descarga de software, visualización de instantánea y SPS

Básicamente, los módulos ya conocidos de TIS 2000 se han reutilizado e integrado en la tecnología web.  Por eso suponemos aquí que no hay necesidad de familiarizarse con los mismos.

6. Asistencia

Si tuviera problemas durante la instalación o ejecución del software, puede ponerse en contacto con el Centro de Asistencia al Distribuidor (DAC).

7. Feedback

Si detecta errores durante el uso de GlobalTIS, hay una forma sencilla de informarnos de dichos errores.  Hay incorporado un formulario de comentarios, relleno ya con información contextual, que sólo tiene que completar con una descripción del error específico y sus datos de contacto.  Sólo tiene que hacer clic en el botón 'Enviar' y automáticamente se nos remitirá un mensaje de correo electrónico.  Ya no tiene que crear un mensaje por separado o enviarnos un fax.

8. Desinstalación

Si desea quitar GlobalTIS de su ordenador, hay 2 formas posibles:

  1. Se puede ejecutar el procedimiento específico de desinstalación desde 'Inicio' - 'Programas' - 'GlobalTIS' - 'Desinstalar GlobalTIS'
  2. El software también se puede desinstalar desde 'Inicio' - 'Configuración' - 'Panel de control' - 'Agregar o quitar programas'.  Debe buscar la entrada 'GlobalTIS' y hacer clic en el botón 'Cambiar/Quitar'.

Se dejará alguna información en el sistema sobre la anterior configuración de la instalación.   Por ejemplo, la información que introdujo y recibió durante el proceso de registro.   Dicha información puede reutilizarse y facilitar una nueva instalación de GlobalTIS.   Pero si selecciona la opción 'Ignorar ajustes antiguos' durante una nueva instalación de GlobalTIS, se perderán todos los datos de registro y tendrá que repetir el procedimiento de licencia.

 

9. Diferencias con TIS 2000

  • El funcionamiento se controla mucho más mediante iconos.
  • La marca de vehículos sólo tiene que definirse una vez después de la instalación de TIS2Web.
  • La página de inicio contiene menos símbolos que los habituales en TIS 2000.
Inicie TIS2Web, haga clic en el icono de 'Configuración', situado en la esquina superior derecha, y seleccione una de las entradas disponibles.   Permanecerá activada incluso después de cerrar el programa.

Las sugerencias más importantes se pueden encontrar en la función 'Ayuda', situada en la esquina superior derecha.
