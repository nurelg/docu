Remarques générales

Le GDS 2 doit être utilisé par les employés de maintenance qualifiés pour effectuer des diagnostics et réparer les systèmes électroniques des automobiles. Nous nous sommes efforcés de fournir des informations techniques complètes et exactes en fonction des informations de maintenance de l'usine disponibles à la date de publication. Toutefois, des modifications peuvent être effectuées à tout moment sans préavis.

Pour vous familiariser avec GDS 2 et ses fonctions, lisez les guides d'utilisation avant d'utiliser GDS.

GDS2 propose les fonctionnalités suivantes en plusieurs langues :

  • Lecture de codes, statut de code, données arrêt sur image et codes de suppression.

  • Lecture de NIV, numéros de pièces ECU et numéros d'applications.

  • Enregistrement, stockage et relecture de données stockées.

  • Contrôle et surveillance de fonctions de contrôle de sortie.

  • Fonctions Configurer et Réinitialiser.

  • Enregistrement et affichage de l'historique des précédentes sessions de diagnostic véhicule.

Important:
La programmation SPS des contrôleurs de véhicule n'est pas prise en charge dans GDS. Pour la programmation SPS Pass Through avec la MDI, utilisez l'application TIS2Web / GlobalTIS SPS.

Installation de GDS 2

Présentation des composants GDS 2 :

Le logiciel GDS 2 contient les modules suivants qui seront installés initialement et mis à jour régulièrement à certaines périodes.

  • GDS 2 Base System
    Logiciel de base GDS 2 installé sur le PC

  •  Livrables GDS 2 : 
    Packages de données de diagnostics pour les modèles de véhicules qui exécutent les diagnostics avec les contrôleurs de véhicule.

  • Logiciel MDI (application séparée):
    Application pour la mise à jour et la configuration de l'unité MDI (Multiple Diagnostic Interface)


 Schéma général des composants de GDS 2 : 

Installation :

GDS2 peut être installé directement en tant que téléchargement dans TIS2Web / Global TIS.

Important:
Un minimum de 2 Go de RAM est nécessaire pour exécuter GDS 2. Les durées de téléchargement et d'installation dépendent de la vitesse de connexion internet disponible pendant l'opération de téléchargement. L'application GDS 2 doit être chargée individuellement sur chaque PC du magasin. Une fois l'installation initiale de l'application effectuée, GDS 2 s'exécutera soit en mode indépendant lorsqu'elle est lancée depuis l'icône du bureau du PC, soit à partir de TIS2Web.

Installation de GDS 2

  • Le PC doit être démarré en mode d'administration.

  • Ouvrez l'application TIS2Web. Cliquez sur 'Continuer'.

  • Une fenêtre de la version de l'application s'affiche. Cliquez sur 'OK'.

  • Sur la page d'accueil de TIS2Web, sélectionnez GDS 2.   [I].

  • Sélectionnez 'Démarrer GDS 2' dans l'écran 'AVANT DE DÉMARRER'.

  • Une fenêtre contextuelle apparaît avec une barre de mise à jour des données. L'installation de GDS 2 peut nécessiter plusieurs étapes.

  • Une fois GDS 2 installé, sélectionnez la langue appropriée et cliquez sur OK. La fenêtre principale de l'application s'ouvre à côté de [II]. Un logo GDS 2 figure également désormais sur votre bureau. La version du logiciel GDS 2 et la version globale GM s'affichent dans le coin inférieur gauche de la fenêtre (12).

  • Vous devez installer le package de diagnostic avant d'utiliser l'application. Si aucun package de diagnostic n'est installé, le message 'Aucun package de diagnostic installé dans l'application' apparait dans le menu principal de GDS 2. Voir Téléchargement et mise à jour de packages de diagnostic de GDS 2, ci-dessous.

Installation à partir de TIS2Web / Global TIS :

Important:
Au cours de l'installation depuis TIS2Web, 50 Mo de données environ sont téléchargées via Internet. En fonction de la sécurité de votre ordinateur, il est possible que votre parefeu requière une reconfiguration afin d'utiliser les fonctions de Tis2Web.
Important:
Les mises à jour décrites nécessitent de faire fonctionner le PC en mode administration.

Système de gestion des licences GDS 2 (Lease)

Le système de contrôle de licence GDS 2 empêche l'utilisation non autorisée de GDS.  Pour renouveler la licence GDS 2, GDS 2 doit être démarré à partir de TIS2Web ou Global TIS. La licence GDS 2 doit être renouvelée au bout de 7 jours si l'application n'est utilisée qu'en mode Standalone pendant cette durée.

Si le renouvellement de licence de GDS 2 échoue, le message d'erreur "Aucune licence valide trouvée" [III] invite l'utilisateur à lancer l'application GDS 2 à partir de TIS2Web / Global TIS pour le renouvellement de licence. Si l'accord de licence échoue, cliquez sur le Client de diagnostic (2)[I-2], supprimez la clé d'application et lancez l'application. Si les difficultés persistent, assurez-vous que l'ordinateur est en mode admin en retournant sur le bureau, en cliquant avec le bouton droit sur Raccourci GDS 2, et en sélectionnant 'Exécuter en tant qu'administrateur'.

Téléchargement et mise à jour des livrables GDS 2

Les Packages de diagnostic de GDS 2 sont des applications de diagnostic contrôlant les services de communication et de diagnostic entre GDS et le contrôleur de véhicule.

Téléchargement et mise à jour des Packages de diagnostic de GDS 2.

  • Cliquez sur Gestion des packages de diagnostic (2) pour installer un package de diagnostic. [II-2].

  • Le ou les packages de diagnostic valides apparaissent sous le titre Mettre à jour les packages de diagnostic. [IV].

  • Sélectionnez le package de diagnostic (1) que vous souhaitez télécharger depuis la liste à l'écran.  Millésime modèle, Marque et Modèle apparaissent vers le bas de l'écran pour le package placé en surbrillance (2). Lorsque vous avez trouvé le package de diagnostic désiré, sélectionnez 'Télécharger la version' (3).

  • Lorsqu'un package de diagnostic est chargé, l'installation s'affiche à l'écran [IV]. La durée de téléchargement varie en fonction de la taille du ou des fichiers de diagnostic installés.

  • Une fois le package de diagnostic installé, GDS 2 se ferme et redémarre automatiquement. Veuillez patienter le temps que la fenêtre GDS 2 apparaisse à nouveau, affichant la fenêtre de démarrage. Cela peut prendre un certain temps. N'essayez pas de démarrer manuellement GDS 2 pendant ce temps [I-3]. Le mot 'Actuel' apparait maintenant dans la fenêtre Mettre à jour des packages de diagnostic pour l'application sélectionnée.

Désinstallation de GDS 2:

GDS 2 peut être désinstallé en démarrant les programmes suivants :

Pour Windows 7 :

'Démarrer' (logo Windows 7) - Panneau de configuration - Programmes et fonctionnalités - Désinstaller un programme - Faîtes un double clic sur 'GDS 2' - Sélectionnez 'Oui'.

Pour Windows XP :

'Démarrer' - Programmes - GDS 2 - Désinstaller GDS 2.

Modifier les paramètres de Langue / Régionaux :

Sélectionnez Langue (4) dans l'écran principal de l'application GDS 2. [II-3] . La liste des langues disponibles apparait.  Sélectionnez la langue appropriée et quittez l'écran en sélectionnant 'Précédent' (8).

Mise en route

Avant d'utiliser GDS 2

Il est important d'observer les précautions suivantes :

Information préalable:

  • Pour éviter d'endommager le module véhicule, utilisez uniquement les versions actuelles du logiciel MDI. GDS 2 et Multiple Diagnostic Interface (MDI) doivent correspondre au dernier logiciel. L'échec de mise à jour de MDI avec le dernier logiciel peut entrainer un diagnostic erroné du véhicule.

  • Assurez-vous que la batterie du véhicule est complètement chargée. La tension de batterie pour GDS 2/MDI doit être comprise entre 12 et 14 volts pour un véhicule 12 volts. En raison des contraintes de temps propres au diagnostic d'un contrôleur, installer l'outil d'aide à la programmation SPS EL-49642 (en Amérique du Nord) ou CX Pro 25/CX Pro 50 (en Europe, distribué par SPX) pour gérer la tension système. Une tension stable de la batterie est critique lors de la programmation. Toute variation, pointe, surtension ou perte de tension interrompt la programmation.

  • Si l'outil d'aide à la programmation SPS EL-49642 ou CX Pro 25/CX Pro 50 n'est pas disponible, NE PAS connecter un chargeur de batterie ordinaire. Brancher un assistant batterie ou une batterie d'appoint 12 V chargée non branchée à une alimentation de CA.

  • Vérifiez que les câbles de connexion sont branchés correctement.

  • Lors de l'utilisation de l'ordinateur portable / autre appareil d'affichage pour GDS 2, s'assurer que le bloc d'alimentation est correctement connecté. S'il est alimenté par un cordon d'alimentation à courant alternatif, s'assurer que l'ordinateur reçoit l'alimentation. Si l'ordinateur portable fonctionne avec sa source d'alimentation interne (batterie), s'assurer qu'elle est correctement chargée avant d'utiliser le programme GDS 2.
Important:
Pour réduire la perte potentielle du signal, configurez MDI pour utiliser l'option de communication la plus stable sur votre site. Vous pouvez choisir USB, Ethernet ou Wireless Ethernet.

Procédure de démarrage GDS 2

Démarrer GDS 2

GDS 2 peut être démarré dans 'TIS2Web/Global TIS' en cliquant sur l'icône 'GDS 2' [I-1]. Vous pouvez également démarrer GDS 2 en mode autonome via le menu Démarrer de Windows.

Sélections de navigation

Près du bas de chaque fenêtre de GDS 2 figurent cinq sélections principales de navigation disponibles [II-4]. Les sélections non actives pour la fenêtre courante s'affichent en gris. Les sélections ont les fonctions suivantes :

Précédent (7) vous renvoie à la fenêtre précédente
Réaction (8) Cette sélection sera disponible ultérieurement. Cette sélection (8) vous donne l'opportunité de formuler une réaction relative à tout problème que vous pourriez rencontrer avec l'application, ou des suggestions d'amélioration que vous souhaiteriez voir. Nous serons également heureux de lire tout commentaire relatif à ce qui fonctionne bien. Lorsque vous décrivez des problèmes, veillez à être aussi précis que possible. Expliquez exactement ce que vous faisiez lorsque le problème est apparu et si le problème peut se produire à nouveau.
Accueil (9) vous redirige vers la fenêtre principale de l'application
Module (10) vous renvoie vers le menu précédent pour le véhicule sélectionné
Entrée (11) vous permet d'effectuer une sélection

Saisie des informations de véhicule/MDI

  1. Sélectionnez l'icône Diagnostics (1) dans l'écran principal de l'application [II-5].

  2. Le menu Sélection de véhicule apparait.

  • [V] Sélection MDI: Toutes les MDI disponibles dans les réseaux chez le distributeur s'affichent. Vous pouvez sélectionnez la MDI que vous utilisez pour effectuer les diagnostics de véhicule. Pour les connections MDI, mettez en surbrillance MDI (1) en cliquant dessus et sélectionnez 'Connecter'. Choisissez 'Sélectionner le dispositif' pour détecter les nouveaux MDI sur les réseaux disponibles. Pour continuer GDS 2 sans MDI connecté, sélectionnez 'Naviguer sans périphérique' (1) [VI]. Une fois effectuée la connexion initiale à un MDI par un PC spécifique, les connexions suivantes seront effectuées automatiquement lorsque le MDI spécifique est disponible.

  • [VI] Entrée NIV: À supposer que le MDI est connecté au véhicule, le numéro NIV (2) est automatiquement reçu par le logiciel GDS 2. Dans le cas contraire, le Numéro d'identification de véhicule (2) peut être saisi manuellement (nécessite une connexion MDI). Une fois le NIV saisi, les marque, modèle et millésime du modèle du véhicule s'affichent à l'écran. Le NIV peut être également lu en sélectionnant 'Lire le NIV' (4), puis 'Décoder le NIV' (3). 'Horodateur' (5) indique la date et l'heure de visualisation/enregistrement des données. Pour continuer l'opération de diagnostic, sélectionnez 'Entrée' (6).

  • Sélection identification véhicule: Si vous ne souhaitez pas saisir de NIV, les marque, modèle et millésime modèle du véhicule peuvent être sélectionnés par l'intermédiaire d'une série d'écrans. Vous avez également la possibilité de sélectionner plusieurs options de diagnostic, selon la ou des fonctions que vous souhaitez explorer. Diagnostics de module, Codes de problème de diagnostic, Informations d'identification, Fonctions de contrôle, Fonctions Configurer/Réinitialiser, etc.

  • [VII] Historique des diagnostics du véhicule: Si un véhicule a été précédemment diagnostiqué avec GDS 2, LE NIV et les informations associées seront conservées au bas de l'écran. Il est possible d'afficher et de conserver des sessions entières de diagnostic de plusieurs NIV. Mettez en surbrillance le véhicule que vous souhaitez diagnostiquer (1) et sélectionnez 'Entrée' (2).

Diagnostics

Afficher des données de diagnostics

Pour saisir des diagnostics, sélectionnez l'icône Diagnostics (1) dans l'écran principal de l'application [II-6].

L'application Afficher des données de diagnostics donne le statut en temps réel des données et signaux du contrôleur de véhicule. Dans Afficher des données de diagnostics de GDS 2, les listes de données disponibles sont disponibles dans un menu déroulant [VIII].

Suspendre la sélection (1): Cliquez sur cette sélection pour suspendre l'affichage des données.
Flèche haut (2): Cliquez sur la flèche haut pour faire défiler la barre en surbrillance d'un paramètre vers le haut. Sélection non disponible du premier paramètre.
Flèche bas (3): Cliquez sur la flèche bas pour faire défiler la barre en surbrillance d'un paramètre vers le bas.
Loupe plus (4): Cliquez sur la loupe plus pour augmenter la taille des données affichées en surbrillance
Loupe moins (5): Cliquez sur la loupe moins pour réduire la taille des données affichées en surbrillance
Icône œil (6): Cette sélection ouvre un nouveau groupe d'icônes vous permettant de modifier les propriétés d'affichage des données graphiques [IX].
Interrupteur (7): Cliquez sur l'interrupteur pour permuter l'affichage de données entre unités britanniques et unités métriques.
Cadenas (8): Déplace les données en surbrillance au sommet de la liste de données.
Icône graphique (9):  Marque les points de données (avec des hachures) sur les graphiques linéaires de données.
Ajouter un signet de capture instantanée (10): Peut être utilisé pour 'marquer' les données comme signet afin de les retrouver ultérieurement.
Onglet graphique linéaire (11): Peut être utilisé pour créer un graphique de données précédemment conservées de la liste de données sélectionnées.
Module (12): Peut être utilisé pour revenir à la liste des modules de véhicule disponibles.
 +  Propose des sélections de direction supplémentaires pour les affichages de graphiques [X].
La Barre de défilement est située entre l'affichage graphique et le menu déroulant de sélections et vous permet de naviguer vers le haut ou le bas dans la liste de données.

Écran DTC

Lorsque Codes de problème de diagnostic (DTC) est sélectionné, un écran de résumé contenant le Nom du module de contrôle (1), le Statut du module de contrôle (2), et le Nombre de DTC (3) s'affiche. [XI].

Sélection de détails

Si Détails [XI] (4) est sélectionné, un écran de DTC détaillé apparait [XII]. Module de contrôle (1), DTC (2), Octet de symptôme (3), Description (4), Description de symptôme (5), et Octet d'état (6) s'affichent tous au sommet du graphique. Au bas du graphique s'affichent la Catégorie (7) et la Valeur décodée (8). Précédent (9) vous redirige vers l'affichage des DTC.

Actualiser les DTC

Si Rafraichir [XII] (10) est sélectionné, GDS 2 demande une mise à jour des DTC au contrôleur.

Effacer les DTC 

Si Effacer les DTC [XII] (11) est sélectionné, un écran d'affichage des DTC apparait [XIII].

Utilisez Ajouter (1) ou Tout ajouter (2) pour sélectionner les ECU que vous souhaitez supprimer. Certaines unités ECU ne peuvent être supprimées qu'en les supprimant toutes. Dans ce cas, elles sont toutes sélectionnées. Sélectionnez Supprimer (3) pour démarrer l'opération de suppression des ECU sélectionnés. Sélectionnez Tout supprimer (4) pour démarrer l'opération de suppression de tous les ECU. Sélectionnez OK (5) pour supprimer les ECU sélectionnés. Sélectionner Annuler (6) si vous ne désirez pas supprimer d'ECU. Si OK est sélectionné, un message de suppression des DTC apparait. Le ou les codes supprimés disparaissent au prochain rafraichissement.

Sortie de l'application DTC

Quittez l'application DTC en sélectionnant Précédent puis Accueil.

Important:
Seule la sélection de Fermer l'application (6) vous permet de quitter complètement l'Application de diagnostic [II-7]. Vous ne pouvez pas quitter GDS 2 lorsque des applications de diagnostics sont actives.  Pour quitter GDS 2, revenez à la page de l'application GDS 2 en sélectionnant Accueil, lorsque disponible.

Données GDS 2 conservées (capture instantanée)

Dans GDS 2, il n'existe pas de capture instantanée conventionnelle comme c'était auparavant le cas. GDS 2 sauvegarde l'ensemble des 20 derniers NIV et données de diagnostic pour consultation ultérieure. Lors de la visualisation de données en direct, l'utilisateur a la possibilité de 'marquer' les données avec des signets. Pour afficher les données précédemment conservées :

  • Sélectionnez Consulter les données conservées (3) dans l'écran d'accueil de GDS [II-8].

  • Sélectionnez Exporter la session (1) [XIV] si vous souhaitez envoyer cette session à l'assistance clientèle, etc. sélectionnez cette option. Sélectionnez la session que vous souhaitez exporter. Ensuite, sélectionnez le répertoire cible dans lequel vous souhaitez l'enregistrer. Sélectionnez Ouvrir et le fichier est enregistré. Ce fichier peut être envoyé dans un e-mail, etc.

  • Choisissez Importer une session (2)[XIV] si vous souhaiter importer un fichier capture instantanée dans GDS 2, sélectionnez cette option. Le fichier capture instantanée sera envoyé vers un répertoire pour être lu dans GDS 2.

  • Sélectionnez Supprimer (3) [XIV] pour supprimer une session.

  • Sélectionnez Capture instantanée (4) [XIV] pour afficher l'événement capture instantanée en surbrillance.

Fonctions de configuration et de réinitialisation :

Les fonctions de configuration et de réinitialisation sont des fonctions de contrôle spécifiques qui doivent être exécutée lorsque les actionneurs systèmes sont remplacés ou doivent être réinitialisés. Pour plus d'informations, voir le manuel de maintenance approprié du système.

Dépannage

Problèmes de connexion MDI

Si les connexions MDI sont instables ou non correctement connectées au véhicule/ordinateur, un message de perte de communication s'affiche. Procédez comme suit :

  • Assurez-vous que le logiciel MDI Manager ne fonctionne pas en même temps que GDS 2.

  • Assurez-vous que le MDI est correctement connecté au véhicule et à l'ordinateur.

  • Assurez-vous qu'une tension suffisante est disponible pour que l'interface véhicule MDI fonctionne correctement.

  • Utilisez un 'câble USB en bon état' reconnu d'une autre unité MDI, si disponible. Confirmez toutes les connexions et essayez à nouveau.

  • Utilisez un 'câble de connexion MDI DLC en bon état' reconnu d'une autre unité MDI, si disponible. Confirmez toutes les connexions et essayez à nouveau.

  • Si l'incident persiste, contactez le Customer Support.

  • Vérifiez que le pare-feu ne bloque pas les communications GDS 2/MDI.

Important:
Le logiciel MDI Manager ne peut être en cours de fonctionnement lorsque l'on tente d'utiliser GDS 2.
