1. 計劃與背景

1.1 簡介

在目前的歐寶汽車（Opel）合作者服務范圍內，TIS 2000是一重要資訊資源。  以DVD光碟反復更新發布形式提供技術文件、測試裝置軟體以及用於電子控制模組（ECU）等的軟體。

TIS 2000正在改進中，將作為新應用程式通過互聯網提供。  其優點將顯而易見：資訊及時更新，避免安裝問題。  一個進入互聯網服務方向的中間步驟是先有這個新的應用程式，但是還在DVD光碟上，為當地安裝而設計。  透過如此操作，減少對互聯網使用的依賴性。

此外，過去的TIS 2000已改變，其方式是壓縮資訊目錄，以支持更快進入到特定文檔。  您現在將可找到一個和諧化的目錄，其中您不僅可看到維修與拆卸說明，還可看到可實用的“現場現場排除故障”、“檢查步驟”和“電路圖”。

2. 安裝（服務器或獨立電腦）

和TIS 2000一樣，這個DVD光碟需要安裝到服務器上，或者在本地的PC電腦（獨立電腦Standalone）上。  但是，也有一些處理的不同，介紹如下。  另一個基本不同點是需要登記注冊。  這表示GlobalTIS只可以在用戶成功確認他是獲得授權的用戶之後才可以使用。  有關細節請參考條款2.1.4。

2.1  安裝前：

  • 要執行安裝或在之後要執行更新的用戶必須具備此電腦的管理權。
  • 開始安裝前，請確認您的電腦上的微軟視窗系統Windows已經用最新的“維護套件”做了更新昇級。
  • 不可存在GlobalTIS的安裝目錄，或此目錄必須為空白。  目錄的名稱不可包括特別字母。
  • 在您有一個硬體鑰匙（HWK）以USB類型連接到電腦上時，在此安裝過程中您必須拆除此鑰匙。  否則，微軟視窗系統Windows會自動安裝駕駛員軟體，這會影響到以后的操作。  那麼就會裝上一個錯誤驅動器。

2.2 安裝中：

  • 此系統將會詢問所用語言。 此選擇只有關此安裝步驟。 “用戶界面”和文檔的語言設定是不不同方式限定。
  • 本系統自行選擇入口，例如“轉換庫服務器入口”。 您不應該更改這些選擇，除非您有意要改變它們。
  • 本系統自動選擇一個主人姓名。這應會在一個獨立PC電腦上留下一個“本地用戶戶主”，或者改變到服務器上的電腦名稱（請看您的桌面上的“我的電腦”，點擊右鼠標，選擇電腦名稱，選擇“更改”按鍵，就可以獲得正確的電腦名稱）。
  • 此系統將會自動選擇“郵件服務器”。  此功能是用在您要把回饋意見直接從此電腦發送出而不需要運行用於傳真的打印。  對於每個默認設置，這些數值都不必更改。

2.3  在安裝後，但在首次啟用前：

在安裝完成後，用戶需要通過“開始”-“程式”-“GlobalTIS”-“GlobalTIS”的路徑手動啟用該應用程式。

3. 登記注冊：

只有獲得授權的用戶可以用GlobalTIS程式工作。  授權可以通過一個登記程序而獲得。

在過去，使用者可以在其HWK轉換之後用GlobalTIS工作。 其序列號碼會自動成為授權（許可證鑰匙）。 因為安全原因，這個授權程序現在不能再繼續使用，需要盡快更新。

  • 聯網登記
  • 手動登記

各個帶有*) 標記的文字輸入欄目都必須用所要求數據完成填寫。  在本表格終結處，用戶可以選擇他是否要上網登記注冊還是手動登記注冊。  在上網登記注冊情況下，用戶相關數據是靠電子信件方式傳送到一個中央位置，在那里產生一個準證鑰匙，然後再送回客戶。  本準證鑰匙是一個編成代碼的編號，必須輸入到“準證鑰匙”的欄目內。

在手動登記注冊情況下，本系統自動產生一個PDF格式的文檔。  可以作為電子郵件附件發到所給地址，但是也可打印出來傳真發送。  但是最好用電子郵件，因為可以處理得更快。

3.1  對於一個銷售成品的訂用權

在登記注冊過程中，用戶將獲得剛好用於1個銷售成品的訂用權。  對於每一個增加的銷售成品，登記注冊需要通過再次履行登記注冊手續才能延長。  在訂用期限延長之後，需要關閉GlobalTIS再重新開始，才能顯示這些新的應用程式圖標。

3.2  訂用權改變

因此，如果要求針對增加的銷售成品的訂用權，現存登記注冊就必須修正。  要做到這點，啟動GlobalTIS應用程式，然後點選標簽“管理（Admin）”  在此您可選擇一個新的訂用權，然後按手動或者上網登記注冊的按鈕。

如果您沒有足夠的獲得授權的用戶許可，則同樣程序是非常必要的，這樣用戶數需要增加。  此數字限制用戶同時連接到一個服務器上的數量。  在多數情況下，此數字不需要修正。

4. 更新昇級GlobalTIS

在GlobalTIS新版本用DVD光碟發布時，它也代表對現有安裝的一次更新昇級。  把該DVD光碟放入電腦的DVD光碟驅動器。  如果該電腦上的“自動播放”選項在啟用狀態，則更新昇級步驟將自動開始。  如果這個功能沒有在您的系統上啟動，您需要從GlobalTIS DVD的根源目錄手動啟動setup.exe程序。  GlobalTIS 不能自動搜索DVD驅動器內的更新。

5. 起動該應用程式

此應用程式可以通過點擊“開始”選單內的進入路徑起動。  找到默認的進入標識的路徑是“開始”-“程式”-“GlobalTIS”-“GlobalTIS”

5.1  啟動一個代理（Client）

如果要在一個客戶電腦上起動該應用程式，所有您需要的工具就是互聯網瀏覽器“Internet Explorer”。  在地址欄內，必須填寫服務器名稱，例如：http://server-name:port/tis2web.

例如： http://asterix:9080/tis2web

“入口”通常為“9080”，或者是在安裝中當需要定義“Tomcat”的入口時已經輸入過的數值。
此地址應該在您的瀏覽器的“收藏”欄內儲存為書簽。  GlobalTIS只能和微軟的互聯網閱讀器（Microsoft Internet Explorer）一起使用。

5.2 啟動一個服務器

一旦啟動服務器，您就需要用一個用戶名登錄。  在首次啟動時，您必須輸入一個新的您選擇的新用戶。  一旦您再次啟動，您可以選擇這個用戶，或者輸入一個新用戶。

5.3  Java 功能執行時間環境

這是針對模組而要求的功能

  • 軟體下載
  • 維修編程系統（SPS）
  • 安全途徑
  • 快照
  • Tech2 檢視
  • Techline列印

如果此電腦已安裝有高於1.4x的Java版本，那麼，當GlobalTIS應用程式在啟動時，該Java應用程式將自動下載。  此情況只發生一次，或者在服務器上有一個修正版本之後發生。
如果還沒有安裝Java應用程式，或者此版本較舊，那麼就需要先安裝。  Java版本1.5.0_11可以從“幫助”欄下載，其部分為： 一般說明 -  下載應用程式

5.4  Java Proxy Settings（代理設定）

在Java應用程式不起動情況下，那麼就有必要對“Java Web Start”（網絡起動）應用程式配置做更改。 代理設定需要關閉停用。
“文檔”-“個人設定”-“一般”。 針對“代理”（Proxies）選擇“否”（None）。

5.5  屏幕設定

要支持有效而方便地處理該軟體，最優化屏幕設定是廣角設定。  基本上，有2個選擇需要注意：

針對文檔區域值設定一個合適的尺寸。  選擇標記“設定”，更改參數“顯示區域高度”，直到此文檔伸展跨越整個頁面。

通過按功能鍵“F11”更改互聯網瀏覽器（Internet Explorer）顯示模式，改到“全部屏幕”。

“來回轉換顯示器”按鈕可以在兩者間選擇：文檔旁的邊靠邊的目錄顯示模式，或者只顯示文檔內容的模式。  這可幫助最優化使用可用屏幕空間。  來回轉換按鈕在圖標欄內可以找到和使用，如下圖所示：

5.6  語言設定

GlobalTIS應用程式的語言是由瀏覽器設定來決定的。  要更改語言，可以打開瀏覽器上的“工具”選單，然後選擇“互聯網選項”，然後選擇按鈕“語言”……  遵照瀏覽器說明操作。
要啟用這些更改，用戶需要再次登錄到這個應用程式。

5.7  瀏覽器設定

目前版本對於至少為1024 x 768皮克斯而文字大小“較小”的屏幕解析度做了優化（互聯網瀏覽器Internet Explorer的路徑為“檢視”-“文字尺寸”）  如果您意圖使用不同設定，有一翻閱欄可能出現在屏幕邊緣，您必須用來看到整個文件。

5.8  分段安排

在TIS 2000中，有一個功能叫做“分段安排”。  它可以允許幾個用戶用TIS 2000工作，但是是在獨立而互不影響的工作環境中。  所有設定，例如車輛類型、車型年及搜索標準等，都是可以保存的，其他用戶可以獲得他們的資訊而不干預。

GlobalTIS沒有同樣技術，但是有一相似的機制。  如果另一用戶需要用這個系統工作，但是不要重新設定所有之前定義的設定，他只需要用互聯網瀏覽器Internet Explorer打開另一個視窗，然後用自己個人身份號碼登錄即可。


5.9  前進與後退功能按鈕

互聯網瀏覽器（Internet Explorer）的前進與後退功能按鈕在GlobalTIS應用程式操作中沒有作用。  因此，不必使用這些按鈕。  相反，在圖標欄內有一個明顯的符號。

 

5.10  軟體下載、快照顯示器及SPS

基本上，TIS 2000的已知名模組都重新使用，並整合在網路技術中。  因此，估計沒有進一步熟悉操作的需要。

6. 支持

如果您在安裝中或者操作該軟體時遇到問題，您可把這些問題報告銷售代理輔助中心（DAC）尋求支持。

7. 反饋

如果您在操作GlobalTIS應用程式時發現缺陷，有一個向我們報告的簡單方式。  有一個內建回饋表已填好相應資訊，您只要添加一個故障說明和您的聯絡數據就可。  所有您要做的就是點擊“發送”按鈕，就可自動產生一個給我們的電子郵件。  您不再需要另外製作一個郵件或者發給我們傳真。

8. 取消安裝

在您要把GlobalTIS應用程式從您的電腦上刪除的情況下，有2種方式可以使用：

  1. 可以通過“開始”-“程式”-“GlobalTIS”-“取消安裝GlobalTIS”的路徑起動一個特定取消安裝步驟。
  2. 也可以通過“開始”-“設定”-“控制面板”-“增加或刪除程式”的路徑來刪除此軟體。  您需要搜索條目‘GlobalTIS’，然後點擊‘改變/刪除’按鈕

有關過去的安裝配置的資訊將留在本系統內。   例如您在登記過程中輸入的或者接收的信息。   它們可以重新使用，使得重新安裝GlobalTIS應用程式比較容易。   但是，如果在一次重新安裝GlobalTIS過程中，您選擇“不管舊的設定”，所有數據就會丟失，您必須重新通過許可證程序。

 

9. 與TIS 2000 的不同

  • 其操作更多依靠圖標來控制。
  • 在安裝TIS2Web之後，此車輛製造只要再定義一次。
  • 開始頁面提供的符號，少於您從TIS 2000中使用的。
啟動TIS2Web，點擊右上角的“設定”圖標，然後選擇其中寫入的可用選項之一。   即使在此程式關閉，它還是保持在此設定狀態。

更重要建議可以在右上角“幫助”功能下找到。
