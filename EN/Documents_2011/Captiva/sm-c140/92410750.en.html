Circuit/System Description

The body control module (BCM) controls the wiper motor based on input from the windscreen wiper washer switch. The BCM monitors the wiper washer switch through three separate signal circuits and a ground circuit. The windscreen wiper switch high signal circuit is used to determine high speed wiper operation, the windscreen wiper switch low signal is used to determine low speed, intermittent, and mist operation through the use of a resistor ladder, and windscreen washer switch signal circuit is used to determine washer operation.

The BCM controls the windscreen wiper motor through two output controls, controlling two relays which determine the desired wiper speed, high or low.

Scan Tool Reference

Control Module References for scan tool information

Circuit/System Verification
  1. Ignition ON, wiper switch in the OFF position, verify the scan tool Windscreen Washer Switch parameter is Inactive.
    2. ⇒If not the specified value, refer to Washers Inoperative .
  2. Wiper switch in the OFF position, verify the scan tool Windscreen Wiper Switch parameter is Off.
    3. ⇒If not the specified value, refer to Windscreen Wiper/Washer Switch Malfunction.
  3. Wiper switch in the OFF position, verify the scan tool Wiper High Speed Switch parameter is Inactive.
    4. ⇒If not the specified value, refer to Windscreen Wiper/Washer Switch Malfunction.
  4. Wiper switch in the LOW position, verify the scan tool Windscreen Wiper Switch parameter is Low.
    5. ⇒If not the specified value, refer to Windscreen Wiper/Washer Switch Malfunction.
  5. Wiper switch in the HIGH position, verify the scan tool Wiper High Speed Switch parameter is Active.
    6. ⇒If not the specified value, refer to Windscreen Wiper/Washer Switch Malfunction.
  6. Wiper switch in the INT position, observe the scan tool Wiper Delay Setting parameter while cycling the delay adjustment switch from long delay to short delay. The parameter should cycle between D1 and D5 as each delay position is selected.
    7. ⇒If not the specified value, test or replace the Windscreen Wiper/Washer Switch Malfunction.
  7. Wiper switch in the LOW position, verify the windscreen wipers are cycling at the low speed.
    8. ⇒If the wipers are not operating at the low speed, refer to Windscreen Wiper Motor Circuit Malfunction.
  8. Wiper switch in the HIGH position, verify the windscreen wipers are cycling at the high speed.
    9. ⇒If the wipers are not operating at the high speed, refer to Windscreen Wiper Motor Circuit Malfunction.
  9. Wiper switch in the INT position, cycling the delay adjustment switch from long delay to short delay. Verify the wiper delay time decreases from long delay to short delay as each delay position is selected.
    10. ⇒If the delay does not change as specified, refer to Windscreen Wiper Motor Circuit Malfunction.
  10. Observe the scan tool BCM Wiper Park Switch parameter. The reading should display Active when the wipers are parked and Inactive with the wipers ON or out of the park position.
    11. ⇒If the Wiper Park Switch parameter is always Active or Inactive, perform the Wiper Park Circuit Malfunction.

Circuit/System Testing

Windscreen Wiper/Washer Switch Malfunction

  1. Ignition OFF, disconnect the wiring harness plug at the Windscreen Wiper/Washer Switch.
  2. Ignition OFF, test for less than 5 Ω between the low reference circuit terminal 3 and ground.
    3. ⇒If greater than the specified range, test the low reference circuit for an open/high resistance. If the circuit test normal, replace the BCM.
  3. Ignition ON, verify the scan tool Windscreen Wiper Switch parameter is Off.
    4. ⇒If not the specified value, test the signal circuit terminal 1 for a short to ground. If the circuit tests normal, replace the BCM.
  4. Verify the scan tool Wiper High Speed Switch parameter Inactive.
    5. ⇒If not the specified value, test the signal circuit terminal 2 for a short to ground. If the circuit tests normal, replace the BCM.
  5. Install a 3 A fused jumper wire between the signal circuit terminal 1 and ground. Verify the scan tool Windscreen Wiper Switch parameter is Active.
    6. ⇒If not the specified value, test the signal circuit for a short to voltage or an open/high resistance. If the circuit tests normal, replace the BCM.
  6. Install a 3 A fused jumper wire between the signal circuit terminal 2 and ground. Verify the scan tool Wiper High Speed Switch parameter is Active.
    7. ⇒If not the specified value, test the signal circuit for a short to voltage or an open/high resistance. If the circuit tests normal, replace the BCM.
  7. If all circuits test normal, test or replace the Windscreen Wiper/Washer Switch.

Wiper Park Circuit Malfunction

  1. Ignition OFF, disconnect the wiring harness plug at the windscreen wiper motor.
  2. Ignition ON, verify the scan tool BCM Wiper Park Switch parameter is Inactive.
    3. ⇒If not Inactive, test the signal circuit terminal 1 for a short to ground. If the circuit tests normal, test or replace the body control module.
  3. Install a 3 A fused jumper wire between the wiper motor park switch signal circuit terminal 1 and ground. Verify the scan tool Wiper Park Switch parameter is Active.
    4. ⇒If not Active, test the signal circuit for an open/high resistance. If the circuit tests normal, replace the body control module.
  4. If all circuit test normal, test or replace the windscreen wiper motor.

Windscreen Wiper Motor Circuit Malfunction

  1. Ignition OFF, disconnect the wiper control relay.
  2. Test for less than 5 Ω between the ground circuit terminal 86 and ground.
    3. ⇒If greater than the specified range, test the ground circuit for an open/high resistance.
  3. Test for less than 5 Ω between the ground circuit terminal 87 A and ground.
    4. ⇒If greater than the specified range, test the ground circuit for an open/high resistance.
  4. Ignition ON, verify a test lamp illuminates between the B+ circuit terminal 87 and ground.
    5. ⇒If the test lamp does not illuminate, test or replace the under bonnet fuse block.
  5. Install a test lamp between the control circuit terminal 85 and the ground circuit terminal 86.
  6. Ignition ON, command the wiper control relay ON and OFF. The test lamp should turn ON and OFF when changing between the commanded states.
    7. ⇒If the test lamp is always ON, test the control circuit for a short to voltage. If the circuit tests normal, replace the BCM.
    ⇒If the test lamp is always OFF, test the control circuit for a short to ground or an open/high resistance. If the circuit tests normal, replace the BCM.
  7. Ignition OFF, disconnect the wiper speed relay.
  8. Test for less than 2 Ω between the control circuit terminal 30 at the wiper control relay and the control circuit terminal 30 at the wiper speed relay.
    9. ⇒If greater than the specified range, replace the under bonnet fuse block.
  9. Connect the wiper control relay.
  10. Test for less than 5 Ω between the control circuit terminal 30 and ground.
    11. ⇒If greater than the specified range, test or replace the wiper control relay.
  11. Ignition ON, verify a test lamp illuminates between the B+ circuit terminal 85 and ground.
    12. ⇒If the test lamp does not illuminate, test or replace the under bonnet fuse block.
  12. Install a test lamp between the control circuit terminal 30 and ground.
  13. Ignition ON, command the wiper control relay ON. The test lamp should turn illuminate.
    14. ⇒If the test lamp does not illuminate, replace the wiper control relay.
  14. Install a test lamp between the B+ circuit terminal 85 and the control circuit terminal 86.
  15. Ignition ON, command the wiper speed relay ON and OFF. The test lamp should turn ON and OFF when changing between commanded states.
    16. ⇒If the test lamp is always ON, test the control circuit for a short to ground. If the circuit tests normal, replace the BCM.
    ⇒If the test lamp is always OFF, test the control circuit for a short to voltage or an open/high resistance. If the circuit tests normal, replace the BCM.
  16. Ignition OFF, connect the wiper speed relay. Disconnect the wiring harness plug at the windscreen wiper motor.
  17. Test for less than 5 Ω between the ground circuit terminal 2 and ground.
    18. ⇒If greater than the specified range, test the ground circuit for an open/high resistance.
  18. Connect a test lamp between the control circuit terminal 4 and ground.
  19. Ignition ON, command the wiper control relay ON and OFF. The test lamp should turn ON and OFF when changing between the commanded states.
    20. ⇒If the test lamp is always ON, test the control circuit for a short to voltage. If the circuit tests normal, test or replace the wiper speed relay.
    ⇒If the test lamp is always OFF, test the control circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the wiper speed relay.
  20. Connect a test lamp between the control circuit terminal 3 and ground.
  21. Ignition ON, manually command wipers to Low Speed.
  22. Command the WIPER HIGH ON and OFF. The test lamp should turn ON and OFF when changing between the commanded states.
    23. ⇒If the test lamp is always ON, test the control circuit for a short to voltage. If the circuit tests normal, test or replace the wiper speed relay.
    ⇒If the test lamp is always OFF, test the control circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the wiper speed relay.
  23. Wipers OFF, verify a test lamp illuminates between the signal circuit terminal 1 and ground.
    24. ⇒If the test lamp does not illuminate, test the signal circuit for a short to ground or an open high resistance. If the circuit tests normal, replace the BCM.
  24. If all circuits test normal, test or replace the windscreen wiper motor.

Component Testing

Windscreen Wiper/Washer Switch Malfunction

  1. Ignition OFF, disconnect the wiring harness plug at the Windscreen Wiper/Washer Switch.
  2. With the wiper switch in the HIGH position, test for less than 2 Ω between the low reference terminal 3 and the signal terminal 2.
    3. ⇒If greater than the specified range, replace the Windscreen Wiper/Washer Switch Malfunction.

    Note: If the switch tests open in any switch position other than Off, test the signal circuits for a short to voltage before replacing the switch.

  3. While cycling the wiper switch through each delay and speed position, test for the specified resistance between the low reference terminal 3 and the signal terminal 1.

Windscreen Wiper Motor

  1. Ignition OFF, disconnect the wiring harness plug at the windscreen wiper motor.
  2. Install a 25 A fused jumper wire between B+ and the control terminal 4. Install a jumper wire between the terminal 2 and ground.
  3. Verify the wiper motor is operating in the low speed mode.
    4. ⇒If the wiper motor is not in the low speed mode, replace the windscreen wiper motor.
  4. Install a 25 A fused jumper wire between B+ and the control terminal 3. Install a jumper wire between the terminal 2 and ground.
  5. Verify the wiper motor is operating in the high speed mode.
    6. ⇒If the wiper motor is not in the high speed mode, replace the windscreen wiper motor.
