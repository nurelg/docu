Captiva
   

Input Shaft Assemble

Tools Required

    • DT-48238 Input 5th Gear Installer
    • DT-48236 Input 3rd/4th Synchroniser Installer

Assembly Procedure

    Important: 

       • Lubricate all components with transaxle fluid before installation.
       • The machined surface faces the needle thrust bearing.

  1. Install the sleeve bearing using an appropriate tool.
  2. Apply Sealer-Retaining (Permatex 620) to the circumference of the input shaft before installing the 5th gear.
  3. Install the 5th gear using the DT-48238 and a hydraulic press.


  4. Install the thrust bearing (2).
  5. Install the two needle bearings (1).


  6. Install the 3rd gear (4).
  7. Install the 3rd gear blocking ring (3).
  8. Install the thrust bearing (2).
  9. Install the thrust washer (1).

    10. Important: 

       • The machined surface of the synchroniser faces the thrust washer.
       • Make sure to align the slots in the synchroniser with the gear blocking ring tabs.

  10. Install the 3rd/4th synchroniser using the DT-48236 and a hydraulic press.
  11. Install the bearing collar using DT-48236 and a hydraulic press.


  12. Install the 4th gear blocking ring (4).
  13. Install the needle bearing (3).
  14. Install the 4th gear (2).
  15. Install the thrust washer (1).
