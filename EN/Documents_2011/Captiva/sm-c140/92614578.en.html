Captiva
   

Manual Transmission Shift Lever Cable Adjustment

Special Tools

DT-527-A Adjuster

For equivalent regional tools, refer to Special Tools .

Removal Procedure
  1. Shift transmission into NEUTRAL.
  2. Remove the front floor console extension panel. Refer Front Floor Console Extension Panel Replacement : Short Body → Long Body


  3. Unlock the shift lever and selector lever cable adjust locks (1), by pushing back the spring force loaded black sliding ring until self-locking.
  4. Unlock the left cable trough the opening at the housing (arrow) using a suitable tool.


  5. Slide the transmission control lever boot upwards.
  6. Press 2 catches (arrows).
  7. Slide blue transmission control lever retainer ring (1) upwards.
  8. Pull trigger (2) upwards.
  9. Remove the transmission control lever knob.


  10. Turn transmission control lever knob (1) 180° and install to transmission control lever.
  11. Push the 2 pins (2) into the 2 holes (3).
  12. Remove the engine control module from battery tray and lay aside.


  13. Block transmission control housing (2) using DT-527-A adjuster (1).

Installation Procedure


    Note: When locking the adjuster locks (1) a defined snap noise should be audible / a snap effect should be felt.

  1. Lock the shift lever and selector lever cable adjust locks (1), by pushing down the two blue locking pieces WITHOUT touching/preloading the fixed shift lever.


  2. Install the transmission control lever knob.

    3. Note: Turn the transmission control lever knob back 180°.

  3. Press the transmission control lever knob during installation completely down. Push trigger (2) upwards.
  4. Slide blue transmission control lever retainer ring (1) downwards.
  5. Remove DT-527-A adjuster from transmission control housing.
  6. Check correct shifting function.
  7. Install the front floor console extension panel. Refer to Front Floor Console Extension Panel Replacement : Short Body → Long Body
