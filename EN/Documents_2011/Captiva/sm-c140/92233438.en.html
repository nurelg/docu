Rear Brake Calliper Replacement

Removal Procedure

Warning: Refer to Brake Dust Warning in the Preface section.

Warning: Refer to Brake Fluid Irritant Warning in the Preface section.

  1. Raise and support the vehicle. Refer to Lifting and Jacking the Vehicle .
  2. Remove the tyre and wheel assembly. Refer to Tyre and Wheel Removal and Installation .


  3. Remove the brake hose fitting bolt (1).
  4. Remove the brake hose fitting (2) from the brake calliper.

    5. Note: Do not reuse the brake hose fitting gaskets.

  5. Remove and discard the brake hose fitting gaskets (3).
  6. Cap the brake hose fitting to prevent brake fluid loss and contamination.


    7. Note: 

       • DO NOT use any air tools to remove or install the guide pin bolts. Use hand tools ONLY.
       • Install an open end wrench to hold the calliper guide pin in line with the brake calliper while removing or installing the calliper guide pin bolt. DO NOT allow the open end wrench to come in contact with the brake calliper. Allowing the open end wrench to come in contact with the brake calliper will cause a pulsation when the brakes are applied.

  7. Using a backup wrench to hold the brake calliper guide pins stationary, remove the brake calliper guide pin bolts (1).
  8. Remove the brake caliper.

Installation Procedure


  1. Install the brake calliper.

    2. Caution: Refer to Fastener Caution in the Preface section.

  2. Using a backup wrench to hold the brake calliper guide pins stationary, install the brake calliper guide pin bolts (1) and tighten to 44 N·m (32 lb ft).


    3. Note: Install new brake hose fitting gaskets.

  3. Assemble the brake hose fitting bolt (1) to the brake hose fitting (2) with 2 new brake hose fitting gaskets (3).
  4. Install the brake hose assembly to the brake calliper and tighten the brake hose fitting bolt to 44 N·m (32 lb ft).
  5. Bleed the hydraulic brake system. Refer to Hydraulic Brake System Bleeding : Manual  → Pressure .
  6. Install the tyre and wheel assembly. Refer to Tyre and Wheel Removal and Installation .
