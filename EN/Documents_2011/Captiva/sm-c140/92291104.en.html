Circuit/System Description

When the ignition switch is in the RUN or ACCY position and the driver switch is pressed in the down position, voltage is applied to the appropriate window motor causing that window to open. When the driver window switch is pulled in the up position, voltage is applied to the appropriate window motor in the opposite direction causing that window to close.

The driver door lock/window switch contains a window lockout switch which disconnects the voltage supply to the rear and passenger door window switches, disabling them. The rear and passenger window switches on the driver door lock/window switch will continue to function normally.

Circuit/System Verification
  1. Ignition ON, command the driver window UP and DOWN by using the window switch on the driver door lock/window switch. The driver window should perform the UP and Down functions.
    2. ⇒If the RPO AA7 driver window does not function as specified, refer to RPO AA7 Driver Window Malfunction.
    ⇒If the RPO ABZ driver window does not function as specified, refer to RPO ABZ Driver Window Malfunction.
  2. Command the passenger window and rear windows UP and DOWN by using the appropriate window switches on the driver door lock/window switch and by using the respective door window switches. Each window should perform the UP and DOWN functions from each switch.
    3. ⇒If the passenger window does not perform the UP/DOWN function, refer to Passenger Window Malfunction.
    ⇒If a rear window does not perform as specified, refer to Rear Window Malfunction.
  3. Disable and enable the rear window switches by using the lockout switch on the driver door lock/window switch. The rear window switches should be enabled and disabled while changing between the commanded states.
    4. ⇒If the windows switches continue to function while disabled, refer to Window Lockout Malfunction.
  4. Command the express UP and DOWN functions by pressing the window switch on the driver door lock/window switch past the first detent and releasing. The driver window should perform the express UP and express DOWN functions while changing between the commanded states.
    5. ⇒If the driver window does not perform as specified, refer to Driver Window Express Malfunction

Circuit/System Testing

RPO AA7 Driver Window Malfunction

  1. Ignition OFF, disconnect the wiring harness plug at the driver door lock/window switch.
  2. Test for less than 2 Ω between the ground circuit terminal listed below and ground.
    3. • RHD terminal 11
    • LHD terminal 14
         ⇒If greater than the specified range, test the ground circuit for an open/high resistance.
  3. Ignition ON, verify that a test lamp illuminates between the ignition circuit terminal listed below and ground.
    4. • RHD terminal 15
    • LHD terminal 10
         ⇒If the test lamp does not illuminate, test the ignition circuit for a short to ground or an open/high resistance.
  4. Ignition OFF, connect the wiring harness plug at the driver door lock/window switch.
  5. Disconnect the wiring harness plug at the driver window motor.
  6. Test for less than 5 Ω between the control circuit terminal 1 and ground.
    7. ⇒If greater than the specified range, test the control circuit for a short to voltage or an open/high resistance. If the circuit tests normal, test or replace the driver door lock/window switch.
  7. Test for less than 5 Ω between the control circuit terminal 2 and ground.
    8. ⇒If greater than the specified range, test the control circuit for a short to voltage or an open/high resistance. If the circuit tests normal, test or replace the driver door lock/window switch.
  8. Connect a test lamp between the control circuit terminal 1 and the control circuit terminal 2.
  9. Ignition ON, command the driver window UP and DOWN by toggling the driver window switch between the UP and DOWN positions. The test lamp should illuminate when commanding the UP and DOWN states.
    10. ⇒If the test lamp is always OFF during either of the commands, test either control circuit for a short to ground. If the circuit tests normal, test or replace the driver door lock/window switch.
  10. If all circuits test normal, test or replace the driver window motor.

RPO ABZ Driver Window Malfunction

  1. Ignition OFF, disconnect the wiring harness plug at the driver door lock/window switch.
  2. Test for less than 2 Ω between the ground circuit terminal listed below and ground:
    3. • RHD terminal 11
    • LHD terminal 14
         ⇒If greater than the specified range, test the ground circuit for an open/high resistance.
  3. Ignition ON, verify that a test lamp illuminates between the ignition circuit terminal listed below and ground.
    4. • RHD terminal 15
    • LHD terminal 10
         ⇒If the test lamp does not illuminate, test the ignition circuit for a short to ground or an open/high resistance.
  4. Ignition OFF, connect the wiring harness plug at the driver door lock/window switch and disconnect the wiring harness plug at the driver window motor.
  5. Test for less than 1 Ω between the ground circuit terminal 3 and ground.
    6. ⇒If greater than the specified range, test the ground circuit for an open/high resistance.
  6. Ignition ON, verify that a test lamp illuminates between the ignition circuit terminal 6 and ground.
    7. ⇒If the test lamp does not illuminate, test the ignition circuit for a short to ground or an open/high resistance.
  7. Verify that a test lamp illuminates between the ignition circuit terminal 3 and ground.
    8. ⇒If the test lamp does not illuminate, test the ignition circuit for a short to ground or an open/high resistance.
  8. Connect a test lamp between the control circuit terminal 2 and ground.
  9. Ignition ON, command the driver door window DOWN by using the driver door lock/window switch. The test lamp should illuminate when commanding the DOWN state.
    10. ⇒If the test lamp does not illuminate, test the control circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the driver door lock/window switch.
    ⇒If the test lamp continuously illuminates, test the control circuit for a short to voltage. If the circuit tests normal, test or replace the driver door lock/window switch.
  10. Connect a test lamp between the control circuit terminal 5 and ground.
  11. Command the driver door window UP by using the driver door lock/window switch. The test lamp should illuminate when commanding the UP state.
    12. ⇒If the test lamp does not illuminate, test the control circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the driver door lock/window switch.
    ⇒If the test lamp continuously illuminates, test the control circuit for a short to voltage. If the circuit tests normal, test or replace the driver door lock/window switch.
  12. If all circuits test normal, replace the driver window motor.

Passenger Window Malfunction

  1. Ignition OFF, disconnect the harness connector at the passenger window switch.
  2. Ignition ON, verify that a test lamp illuminates between the ignition circuit terminal 3 and ground.
    3. ⇒If the test lamp does not illuminate, test the ignition circuit for an open/high resistance.
  3. Ignition OFF, test for less than 5 Ω between the signal circuit terminal 1 and ground.
    4. ⇒If greater than the specified range, test the signal circuit for a short to voltage or an open/high resistance. If the circuit tests normal, test or replace the driver door lock/window switch.
  4. Test for less than 5 Ω between the signal circuit terminal 4 and ground.
    5. ⇒If greater than the specified range, test the signal circuit for a short to voltage or an open/high resistance. If the circuit tests normal, test or replace the driver door lock/window switch.
  5. Connect a test lamp between the signal circuit terminal 1 and the signal circuit terminal 4.
  6. Ignition ON, command the passenger window UP and DOWN by toggling the appropriate window switch on the driver door lock/window switch between the UP and DOWN positions. The test lamp should illuminate when commanding the UP and DOWN states.
    7. ⇒If the test lamp does not illuminate during either of the commands, test either signal circuit for a short to ground. If the circuits test normal, test or replace the driver door lock/window switch.
  7. Ignition OFF, connect the wiring harness plug at the passenger window switch and disconnect the wiring harness plug at the passenger window motor.
  8. Test for less than 5 Ω between the control circuit terminal 1 and ground.
    9. ⇒If greater than the specified range, test the control circuit for a short to voltage or an open/high resistance. If the circuit tests normal, test or replace the front passenger window switch.
  9. Test for less than 5 Ω between the control circuit terminal 2 and ground.
    10. ⇒If greater than the specified range, test the control circuit for a short to voltage or an open/high resistance. If the circuit tests normal, test or replace the passenger window switch.
  10. Connect a test lamp between the control circuit terminal 1 and the control circuit terminal 2.
  11. Ignition ON, command the passenger window UP and DOWN by toggling the passenger window switch between the UP and DOWN positions. The test lamp should illuminate when commanding the UP and DOWN states.
    12. ⇒If the test lamp does not illuminate during either of the commands, test either control circuit for a short to ground. If the circuits test normal, test or replace the passenger window switch.
  12. If all circuits test normal, test or replace the passenger window motor.

Rear Window Malfunction

  1. Ignition OFF, disconnect the wiring harness plug at the appropriate rear window switch.

    2. Note: Verify the rear windows are enabled by checking the window lockout switch on the driver door lock/window switch.

  2. Ignition ON, verify that a test lamp illuminates between the control circuit terminal 3 and ground.
    3. ⇒If the test lamp does not illuminate, test the control circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the driver door lock/window switch.
  3. Ignition OFF, test for less than 5 Ω between the signal circuit terminal 1 and ground.
    4. ⇒If greater than the specified range, test the signal circuit for a short to voltage or an open/high resistance. If the circuit tests normal, test or replace the driver door lock/window switch.
  4. Test for less than 5 Ω between the signal circuit terminal 4 and ground.
    5. ⇒If greater than the specified range, test the signal circuit for a short to voltage or an open/high resistance. If the circuit tests normal, test or replace the driver door lock/window switch.
  5. Connect a test lamp between the signal circuit terminal 1 and the signal circuit terminal 4.
  6. Ignition ON, command the rear window UP and DOWN toggling the appropriate window switch on the driver door lock/window switch between the UP and DOWN positions. The test lamp should illuminate when commanding the UP and DOWN states.
    7. ⇒If the test lamp does not illuminate during either of the commands, test either signal circuit for a short to ground. If the circuits test normal, test or replace the driver door lock/window switch.
  7. Ignition OFF, connect the wiring harness plug at the rear window switch and disconnect the wiring harness plug at the rear window motor.
  8. Test for less than 5 Ω between the control circuit terminal 1 and ground.
    9. ⇒If greater than the specified range, test the control circuit for a short to voltage or an open/high resistance. If the circuit tests normal, test or replace the rear window switch.
  9. Test for less than 5 Ω between the control circuit terminal 2 and ground.
    10. ⇒If greater than the specified range, test the control circuit for a short to voltage or an open/high resistance. If the circuit tests normal, test or replace the rear window switch.
  10. Connect a test lamp between the control circuit terminal 1 and the control circuit terminal 2.
  11. Ignition ON, command the rear window UP and DOWN by toggling the rear window switch between the UP and DOWN positions. The test lamp should illuminate when commanding the UP and DOWN states.
    12. ⇒If the test lamp does not illuminate during either of the commands, test either control circuit for a short to ground. If the circuits test normal, test or replace the rear window switch.
  12. If all circuits test normal, test or replace the rear window motor.

Window Lockout Malfunction

  1. Ignition OFF, disconnect the wiring harness plug at the appropriate rear window switch.
  2. Connect a test lamp between the control circuit terminal 3 and ground.
  3. Ignition ON, command the rear window lockout ON and OFF by pressing and depressing the lockout switch on the driver door lock/window switch. The test lamp should turn ON and OFF when changing between the commanded states.
    4. ⇒If the test lamp is always ON, test the control circuit for a short to voltage.
    ⇒If the test lamp is always OFF, test the control circuit for a short to ground or an open/high resistance.
  4. If all circuits test normal, test or replace the driver door lock/window switch.

Driver Express Malfunction

  1. Ignition OFF, disconnect the wiring harness plug at the driver window motor.
  2. Connect a test lamp between the control circuit terminal 1 and ground.
  3. Ignition ON, command the driver window to express DOWN by pressing and holding the window switch on the driver door lock/window switch. The test lamp should illuminate while commanding the express DOWN state.
    4. ⇒If the test lamp is always ON, test the control circuit for a short to voltage. If the circuit tests normal, test or replace the driver door lock/window switch.
    ⇒If the test lamp is always OFF, test the control circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the driver door lock/window switch.
  4. If all circuits test normal, test or replace the driver window motor.

Passenger/Rear Window Switch

  1. Ignition OFF, disconnect the wiring harness plug at the appropriate passenger or rear window switch.
  2. Test for infinite resistance between the control terminal 8 and terminal 5
    3. ⇒If less than the specified value, replace the window switch.
  3. Test for less than 2 Ω between the control terminal 3 and control terminal 8 with the switch in the UP position.
    4. ⇒If greater than the specified range, replace the window switch.
  4. Test for less than 2 Ω between the control terminal 3 and control terminal 5 with the switch in the DOWN position.
    5. ⇒If greater than the specified range, replace the window switch.

Window Motor

  1. Ignition OFF, disconnect the wiring harness plug at the appropriate window motor.
  2. Install a 20A fused jumper wire between one of the control terminals and 12V. Momentarily install a jumper wire between the other control terminal and ground. Reverse the jumper wires at least two times, the window motor should perform the UP/DOWN functions.
    3. ⇒If it does not perform as specified, replace the window motor.
