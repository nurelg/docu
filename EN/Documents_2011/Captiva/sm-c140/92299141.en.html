Circuit/System Description

Circuit/System Verification
  1. Ignition ON, observe the scan tool Door Ajar parameter with all doors closed. The parameters should all display Off.
    2. ⇒If any parameter displays On, refer to Courtesy Lamps Always On .
  2. Radio on, verify that the front left speaker is operational.
    3. ⇒If the left front speaker is not operational, refer to Diagnostic System Check - Vehicle .
  3. Perform each of the following to verify proper system operation:
    4. • Ignition ON, headlamp ON, verify the scan tool Headlamp Switch parameter is Active.
    • Ignition ON, headlamp OFF, verify the scan tool Headlamp Switch parameter is Inactive.
         ⇒If not the specified value, refer to Headlamp Switch Circuit Test.
  4. Perform each of the following to verify proper system operation:
    5. • Ignition OFF, key in ignition, verify the scan tool Key In Ignition Status parameter is Key In.
    • Ignition OFF, key out of the ignition, verify the scan tool Key In Ignition Status parameter is Key Out.
    ⇒If not the specified value, refer to Ignition Switch Circuit Test.

Circuit/System Testing

Indicator/Multifunction Switch

  1. Ignition OFF, disconnect the wiring harness plug at the indicator/multifunction switch.
  2. Test for less than 1.0 Ω between the ground circuit terminal 7 and ground.
    3. ⇒If greater than the specified range, test the ground circuit for an open/high resistance.
  3. Ignition ON, verify the scan tool Headlamp Switch parameter is Inactive.
    4. ⇒If not the specified value, test the signal circuit terminal 1 for a short to ground. If the circuit tests normal, replace the BCM.
  4. Install a 3 A fused jumper wire between the signal circuit terminal 1 and the ground circuit terminal 7. Verify the scan tool Headlamp Switch parameter is Active.
    5. ⇒If not the specified value, test the signal circuit for a short to voltage or an open/high resistance. If the circuit tests normal, replace the BCM.
  5. If all circuits test normal, test or replace the turn signal/multifunction switch.

Ignition Switch

  1. Ignition OFF, disconnect the wiring harness plug at the ignition switch.
  2. Ignition ON, verify that a test lamp illuminates between the B+ circuit terminal 2 and ground.
    3. ⇒If the test lamp does not illuminate, test the B+ circuit for a short to ground or an open/high resistance. If the circuit tests normal and the B+ circuit fuse is open, test the signal circuit terminal 4 for a short to ground.
  3. Ignition OFF and key out, verify the scan tool Key In Ignition parameter is key out.
    4. ⇒If not the specified value, test the signal circuit terminal 4 for a short to voltage. If the circuit tests normal, replace the BCM.
  4. Install a 3 A fused jumper wire between the signal circuit terminal 4 and the B+ circuit terminal 2. Verify the scan tool Ignition Switch is Key In.
    5. ⇒If not the specified value, test the signal circuit for an open/high resistance. If the circuit tests normal, replace the BCM.
  5. If all circuits test normal, test or replace the ignition switch.

Component Testing

Indicator/Multifunction Switch

  1. Ignition OFF, disconnect the wiring harness plug at the indicator/multifunction switch.
  2. Turn the indicator/multifunction switch to HDLP and test for less than 1.0 Ω between terminal 7 and 1.
    3. ⇒If not within the specified range, test or replace the indicator/multifunction switch.
  3. Turn the indicator/multifunction switch to OFF and test for infinite resistance between terminals 7 and 1.
    4. ⇒If not the specified value, test or replace the indicator/multifunction switch.

Ignition Switch

  1. Ignition OFF, disconnect the wiring harness plug at the ignition switch.
  2. Insert the Ignition Key into the ignition switch and test for less than 1.0 Ω between terminal 2 and 4.
    3. ⇒If not within the specified range, replace the ignition switch.
  3. Remove the Ignition Key from the ignition switch and test for infinite resistance between terminals 2 and 4.
    4. ⇒If not the specified value, test or replace the ignition switch.
