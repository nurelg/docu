Circuit/System Description

The window motors are supplied with B+ voltage which is used to power the motor. When the driver or passenger or rear window switch is pressed, a switch is closed providing ignition voltage in the appropriate circuit in the window motor, the window motor responds by moving the window in the direction commanded.

Diagnostic Aids

A window motor that has not been normalised will no longer perform the express up and express down functions, this may occur during the following conditions:

    • When a window motor has been disconnected
    • When a door harness has been disconnected
    • When the battery has been disconnected or replaced

To normalize the window motor refer to Power Window Reinitialisation

Circuit/System Verification
  1. Ignition ON, command the driver window UP, DOWN, and EXPRESS UP by using the appropriate window switch on the S79D window switch-driver. The driver window should perform the UP, DOWN and EXPRESS functions.
    2. ⇒If the driver window motor does perform as specified, refer to Driver Window Malfunction.
  2. Command the passenger window UP, DOWN, and EXPRESS UP by using the appropriate window switch on the S79D window switch-driver and by using the S79P window switch-passenger. The passenger window should perform the UP, DOWN and EXPRESS functions from each switch.
    3. ⇒If the passenger window motor does perform as specified, refer to Passenger Window Malfunction.
  3. Command the left rear window UP, DOWN, and EXPRESS UP by using the appropriate window switch on the S79D window switch-driver and by using the S79LR window switch-left rear. The left rear window should perform the UP, DOWN and EXPRESS functions from each switch.
    4. ⇒If the left rear window does perform as specified, refer to Rear Window Malfunction.
  4. Command the right rear window UP, DOWN, and EXPRESS UP by using the appropriate window switch on the S79D window switch-driver and by using the S79RR window switch-right rear. The right rear window should perform the UP, DOWN and EXPRESS functions from each switch.
    5. ⇒If the right rear window does perform as specified, refer to Rear Window Malfunction.

Circuit/System Testing

Driver Window Malfunction

  1. Ignition OFF, disconnect the harness connector at the S79D window switch-driver.
  2. Ignition ON, verify that a test lamp illuminates between the ignition circuit terminal 3 and ground.
    3. ⇒If the test lamp does not illuminate, test the ignition circuit for a short to ground or an open/high resistance.
  3. Ignition OFF, connect the harness connector at the S79D window switch-driver.
  4. Ignition OFF, disconnect the harness connector at the M74D window motor-driver.
  5. Test for less than 10 Ω between the ground circuit terminal 3 and ground.
    6. ⇒If greater than the specified range, test the ground circuit for an open/high resistance.
  6. Ignition ON, verify that a test lamp illuminates between the ignition circuit terminal 6 and ground.
    7. ⇒If the test lamp does not illuminate, test the ignition circuit for a short to ground or an open/high resistance.
  7. Connect a test lamp between the signal circuit terminal 1 and ground.
  8. Command the M74D window motor-driver to EXPRESS DOWN by using the window switch on the S79D window switch-driver. The test lamp should illuminate during the EXPRESS DOWN command.
    9. ⇒If the test lamp is always OFF, test the signal circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the S79D window switch-driver.
    ⇒If the test lamp is always ON, test the signal circuit for a short to voltage. If the circuit tests normal, test or replace the S79D window switch-driver.
  9. Connect a test lamp between the signal circuit terminal 2 and ground.
  10. Command the M74D window motor-driver DOWN by using the window switch on the S79D window switch-driver. The test lamp should illuminate during the DOWN command.
    11. ⇒If the test lamp is always OFF, test the signal circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the S79D window switch-driver.
    ⇒If the test lamp is always ON, test the signal circuit for a short to voltage. If the circuit tests normal, test or replace the S79D window switch-driver.
  11. Connect a test lamp between the signal circuit terminal 5 and ground.
  12. Command the M74D window motor-driver UP by using the window switch on the S79D window switch-driver. The test lamp should illuminate during the UP command.
    13. ⇒If the test lamp is always OFF, test the signal circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the S79D window switch-driver.
    ⇒If the test lamp is always ON, test the signal circuit for a short to voltage. If the circuit tests normal, test or replace the S79D window switch-driver.
  13. If all circuits test normal, replace the M74D window motor-driver.

Passenger Window Malfunction

  1. Ignition OFF, disconnect the harness connector at the S79P window switch-passenger.
  2. Ignition ON, verify that a test lamp illuminates between the ignition circuit terminal 3 and ground
    3. ⇒If the test lamp does not illuminate, test the ignition circuit for a short to ground or an open/high resistance.
  3. Ignition OFF, disconnect the harness connector at the M74P window motor-passenger.
  4. Test for less than 10 Ω between the ground circuit terminal 3 and ground.
    5. ⇒If greater than the specified range, test the ground circuit for an open/high resistance.
  5. Ignition ON, verify that a test lamp illuminates between the ignition circuit terminal 6 and ground.
    6. ⇒If the test lamp does not illuminate, test the ignition circuit for a short to ground or an open/high resistance.
  6. Connect a test lamp between the signal circuit terminal 5 and ground.
  7. Command the passenger window UP by using the appropriate window switch on the S79D window switch-driver. The test lamp should illuminate when commanding the UP state.
    8. ⇒If the test lamp is always OFF, test the signal circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the S79D window switch-driver.
    ⇒If the test lamp is always ON, test the signal circuit for a short to voltage. If the circuit tests normal, test or replace the S79D window switch-driver.
  8. Connect a test lamp between the signal circuit terminal 2 and ground.
  9. Command the passenger window DOWN by using the window switch on the S79D window switch-driver. The test lamp should illuminate when commanding the DOWN state.
    10. ⇒If the test lamp is always OFF, test the signal circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the S79D window switch-driver.
    ⇒If the test lamp is always ON, test the signal circuit for a short to voltage. If the circuit tests normal, test or replace the S79D window switch-driver.
  10. Connect a test lamp between the signal circuit terminal 1 and ground.
  11. Command the passenger window EXPRESS DOWN by using the window switch on the S79D window switch-driver. The test lamp should illuminate when commanding the EXPRESS DOWN state.
    12. ⇒If the test lamp is always OFF, test the signal circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the S79D window switch-driver.
    ⇒If the test lamp is always ON, test the signal circuit for a short to voltage. If the circuit tests normal, test or replace the S79D window switch-driver.
  12. Connect the wiring harness plug at the S79P window switch-passenger.
  13. Connect a test lamp between the signal circuit terminal 5 and ground.
  14. Command the passenger window UP by using the S79P window switch-passenger. The test lamp should illuminate when commanding the UP state.
    15. ⇒If the test lamp is always OFF or always ON, test or replace the S79P window switch-passenger.
  15. Connect a test lamp between the signal circuit terminal 2 and ground.
  16. Command the passenger window DOWN by using the S79P window switch-passenger. The test lamp should illuminate when commanding the DOWN state.
    17. ⇒If the test lamp is always OFF or always ON, test or replace the S79P window switch-passenger.
  17. Connect a test lamp between the signal circuit terminal 1 and ground.
  18. Command the passenger window EXPRESS DOWN by using the S79P window switch-passenger. The test lamp should illuminate when commanding the EXPRESS DOWN state.
    19. ⇒If the test lamp is always OFF or always ON, test or replace the S79P window switch-passenger.
  19. If all circuits test normal, replace the M74P window motor-passenger.

Rear Window Malfunction

  1. Ignition OFF, disconnect the harness connector at the appropriate rear S79 window switch.
  2. Connect a test lamp between the control circuit terminal 3 and ground.
  3. Ignition ON, command the rear window lockout ON and OFF by pressing the lockout switch on the S79D window switch-driver. The test lamp should turn ON and OFF with each press of the lockout switch.
    4. ⇒If the test lamp is always OFF, test the control circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the S79D window switch-driver.
    ⇒If the test lamp is always ON, test the control circuit for a short to voltage. If the circuit tests normal, test or replace the S79D window switch-driver.
  4. Ignition OFF, disconnect the harness connector at the appropriate M74 window motor.
  5. Test for less than 10 Ω between the ground circuit terminal 3 and ground.
    6. ⇒If greater than the specified range, test the ground circuit for an open/high resistance.
  6. Ignition ON, verify that a test lamp illuminates between the ignition circuit terminal 6 and ground.
    7. ⇒If the test lamp does not illuminate, test the ignition circuit for a short to ground or an open/high resistance.
  7. Connect a test lamp between the signal circuit terminal 5 and ground.
  8. Command the rear window UP by using the appropriate window switch on the S79D window switch-driver. The test lamp should illuminate when commanding the UP state.
    9. ⇒If the test lamp is always OFF, test the signal circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the S79D window switch-driver.
    ⇒If the test lamp is always ON, test the signal circuit for a short to voltage. If the circuit tests normal, test or replace the S79D window switch-driver.
  9. Connect a test lamp between the signal circuit terminal 2 and ground.
  10. Command the rear window DOWN by using the window switch on the S79D window switch-driver. The test lamp should illuminate when commanding the DOWN state.
    11. ⇒If the test lamp is always OFF, test the signal circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the S79D window switch-driver.
    ⇒If the test lamp is always ON, test the signal circuit for a short to voltage. If the circuit tests normal, test or replace the S79D window switch-driver.
  11. Connect a test lamp between the signal circuit terminal 1 and ground.
  12. Command the rear window to EXPRESS DOWN by using the window switch on the S79D window switch-driver. The test lamp should illuminate when commanding the EXPRESS DOWN state.
    13. ⇒If the test lamp is always OFF, test the signal circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the S79D window switch-driver.
    ⇒If the test lamp is always ON, test the signal circuit for a short to voltage. If the circuit tests normal, test or replace the S79D window switch-driver.
  13. Connect the wiring harness plug at the rear S79 window switch.

    14. Note: Verify the rear window switches are enabled by checking the lockout switch on the S79D window switch-driver

  14. Connect a test lamp between the signal circuit terminal 5 and ground.
  15. Command the rear window UP by using the rear S79 window switch. The test lamp should illuminate when commanding the UP state.
    16. ⇒If the test lamp is always OFF or always ON, test or replace the rear S79 window switch.
  16. Connect a test lamp between the signal circuit terminal 2 and ground.
  17. Command the rear window DOWN by using the rear S79 window switch. The test lamp should illuminate when commanding the DOWN state.
    18. ⇒If the test lamp is always OFF or always ON, test or replace the rear S79 window switch.
  18. Connect a test lamp between the signal circuit terminal 1 and ground.
  19. Command the rear window EXPRESS DOWN by using the rear S79 window switch. The test lamp should illuminate when commanding the EXPRESS DOWN state.
    20. ⇒If the test lamp is always OFF or always ON, test or replace the rear S79 window switch.
  20. If all circuits test normal, replace the rear M74 window motor.
