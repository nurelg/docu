  60. Install the input shaft and output shaft groove ball bearings into the transaxle housing using the DT-48179 handle  (1) and the DT-48226 remover/installer  (2).


  61. Install the needle sleeve flush with the casting in the transaxle housing using the DT-48179 handle  (1) and the DT-48182 remover/installer  (2).

    62. Caution: Refer to Fastener Caution in the Preface section.



  62. Install the reverse shifting blocker (1), the torsion spring (2), and a new thread pin (3) to the transaxle housing and tighten to 6-9 N·m (53-80 lb in).


  63. Install the output shaft roller bearing to the clutch housing.


  64. Install intermediate (1), input (2) and output (3) shaft assembly with the respective selector forks in the clutch housing.
    65. • Install the DT-48232 protector and the input shaft retainer plug.
    • Install the intermediate (1) and input (2) shaft assembly with the respective selector forks.

       Note: Position the Intermediate shaft to the groove ball bearing side.

    • Install the output shaft assembly and the selector fork.
    • After checking the gears engagement, install each the shaft assembly completely.
    • Remove the DT-48232 protector and the input shaft retainer plug.


  65. Install the transaxle housing on the clutch housing.
    66. 65.1. Input and output shaft must engage in the groove ball bearings in the transaxle housing.
    65.2. The Intermediate shaft and the selector forks must sit in their opposing bearings in the transaxle housing.
    65.3. Install the DT-48231 pusher/puller on the transaxle housing.
            • Tighten the four bolts (1).
            • Screw the spindles into the input and output shaft.
    65.4. Draw the input and output shaft evenly into the groove ball bearings in the transaxle housing.
            • Hold the spindles (2) using a suitable tool.
            • Turn via nuts (3) in direction of arrow.
    65.5. Remove the DT-48231 pusher/puller from the transaxle housing.
            • Unscrew the spindles into the input and output shaft.
            • Remove the four bolts (1).


    Note: If large forces are necessary when actuating the DT-48231 pusher/puller separate the transaxle housing and the clutch housing again and check positioning of the shaft and selector forks. Repeat operation.

  66. Shift the transaxle into 4th and 5th gear using a screwdriver in an upward direction.
  67. Install new input and output shaft retainer plugs while holding the transaxle into 4th and 5th gear in an upward direction and tighten.
    68. • The input shaft retainer plug to 120-130 N·m (89-96 lb ft).
    • The output shaft retainer plug to 120-130 N·m (89-96 lb ft).


    Note: Leave area free in order not to block up the oil pipe.

  68. Apply sealant (LOCTITE 5900) to the transaxle housing rear cover.


    69. Note: The oil pipe only installs in one direction. Ensure the oil pipe is completely seated in the gear shafts.

  69. Install the oil pipe and transaxle housing rear cover.


  70. Install the transaxle housing rear cover bolts and tighten to 20-25 N·m (15-18 lb ft).


    71. Note: In line with the Intermediate shaft, use suitable tools to assist in the housing separation.

  71. Remove the clutch housing from the transaxle housing.

  73. Install the idle shift rail bush (2) to the transaxle housing.
  74. Install the drain channel (3) into the transaxle housing.


  75. Install the reversing lamp switch and the washer to the transaxle housing and tighten to 20 N·m (15 lb ft).


  76. Install the shift shaft assembly.
    77. • Apply sealer (LOCTITE 5203) to the transaxle housing to the gear shift cover mating surface.
    • Position the transaxle into neutral.
    • Install the shift shaft assembly (3).
    • Install the select control lever (2).
    • Install the shift retaining bolts and tighten to 20-25 N·m (15-18 lb ft).
    • Install the shift guide bolt (1) and tighten to 20-25 N·m (15-18 lb ft).


  77. Install the differential bearing race and the shim to the transaxle housing using the DT-48179 handle  (1) and the DT-48230 installer  (2).


  78. Install the differential bearing race and the shim to the clutch housing using the DT-48179 handle  (1) and the DT-48230 installer  (2).


  79. Install the magnet into the transaxle housing.


  80. Install the differential into the transaxle housing.


  81. Apply sealer (LOCTITE 5203) to the transaxle housing to clutch housing mating surface.

    82. Note: 

       • the input and output shaft must engage in its bearing in the clutch housing.
       • The gear selector forks sit in their opposing bearings in the clutch housing.

  82. Install the clutch housing to the transaxle housing.


  83. Using a rubber mallet, tap lightly on the transaxle housing mating surface to ensure the bearings are fully seated.


  84. Install the DT-48200 fixing  (1) to transaxle housing.

    85. Note: Adjust the height of the DT-48200 fixing for the zero point setting.

  85. Adjust the zero point setting for each place on the transaxle housing flange surface Using the DT-48200 fixing and the height gauge.


  86. Measure the three places on the output shaft gear thrust washer surface using the height gauge. (average=A)

    87. The height from the transaxle housing flange surface to the thrust washer surface.



  87. Place the clutch housing differential bearing race, the differential, the transaxle housing differential bearing race, and the DT-48241 plate  (1) onto the DT-48242 plate  (2).


  88. Adjust the zero point setting for each place on the transaxle housing differential bearing race surface using the height gauge (1).


  89. Measure the three places on differential ring gear surface using the height gauge (1). (average=B)

    90. The height from the transaxle housing differential bearing race to the differential ring gear surface.



  90. Measure the three places on the DT-48242 plate  (1) using the height gauge. (average=C)

    91. The height from the transaxle housing differential bearing race to clutch housing differential bearing race.



  91. To obtain shim size (1, 2) for transaxle housing shim use the following formula.

    92. S1 (For Both FWD and AWD)=62.92 mm - A - B

  92. To obtain shim size for clutch housing shim use the following formula :
    93. • S2 (For FWD)=156.84 mm - C - S1
    • S2 (For AWD)=158.09 mm - C - S1


  93. Hold the input shaft with the DT-48225 holder  (1).
  94. Draw the Intermediate shaft into the Intermediate shaft groove ball bearing with old retainer plug.

    95. Shift the transaxle into any gear.



  95. Install the clutch housing to transaxle housing bolts and tighten to 20-25 N·m (15-18 lb ft).
  96. Remove the old retainer plug from the intermediate shaft, and install a new retainer plug and tighten to 90-100 N·m (66-74 lb ft).
  97. Remove the DT-48225 holder .


  98. Install the transaxle housing to clutch housing bolts and tighten to 20-25 N·m (15-18 lb ft).


  99. Install a new oil pipe into the intermediate shaft.
  100. Install a new intermediate shaft sealing plug.


  101. Install the O-ring concentric slave cylinder.


  102. After installing the pipe screw, tighten the concentric slave cylinder retaining bolts to 8-12 N·m (71-106 lb in).
  103. Tighten the pipe screw to 16 N·m (12 lb ft).


  104. Install the bush to the concentric slave cylinder pipe.


  105. Install the shift control cable bracket, the washer, and the bolts and tighten to 15-20 N·m (11-15 lb ft).
  106. Install the transaxle into the vehicle. Refer to Transmission Replacement .
