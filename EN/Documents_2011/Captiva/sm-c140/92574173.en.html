Valve Guide Reaming, and Valve and Seat Grinding - LD9

Tools Required

    • KM-571-B Angular Torque Gauge
    • KM-340-1-C Cutter Set

Valve Grind-In


  1. Lubricate the valve seat using a fine-grained paste.
  2. Lift the valve rhythmically from the seat with a commercially available valve grinding tool in order to distribute the paste.


  3. Check the contact pattern on the valve head and in the cylinder head.
  4. Clean the valves, valve guides and the cylinder head.

Valve Grind


  1. Ensure that there is no crater line burns on the valve cone.
  2. The valve may only be reground twice. Do not grind the valve stem end.
  3. Ensure that the angle at the valve face is 45 degrees.
  4. Inspect the assembly height of the inlet valves and exhaust valves.

Valve Guide - Ream


  1. Measure the diameter of the valve guide using KM-571-B and a commercially available inside micrometer.

    2. Important: Valve oversize may already have been fitted in production.

  2. An oversize service code is on the valve guide and the valve stem end. The following table gives the correct size, reamer and production code for each service.
  3. Ream the valve guide from the upper side of the cylinder head to the next oversize.
  4. After reaming, cross out the code and emboss the valve guide with the new code.

Valve Seat - Cut


  1. Place the cylinder head on wooden blocks.
  2. Cut the inlet and the exhaust valve seats using the guide drift KM-340-7 as follows :
    3. • Valve seat -- 45 degree surface using KM-340-13.
    • Upper correction angle -- 30 degree surface using KM-340-13.
    • Lower the correction angle -- 60 degree surface using KM-340-13.
  3. Clean the chippings from the cylinder head.
  4. Inspect the dimension for the valve seat width.
  5. Inspect the assembly height of the inlet valves and the exhaust valves. If the dimension is exceeded, install new valves. Inspect the assembly height again. If the valve assembly height is still too large despite replacing the valves, replace the cylinder head.
