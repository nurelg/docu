Parameter

Expected Value

Definition

Operating Conditions: Ignition ON

Immobiliser Auto Learn Counter

Varies

This displays Counts. This is the number of times the auto learn procedure was carried out by the TDM.

Immobiliser Automatic Learn Timer

Varies

This displays minutes. This is the amount of minutes the TDM took to auto learn.

Battery Reconnect Detected

No

The scan tool displays Yes or No. Yes is displayed if the battery was recently reconnected.

Fleet Vehicle

No

The scan tool displays Yes or No. Yes is displayed if the vehicle is a fleet vehicle.

ECM Fuel Enable

No

The scan tool displays Yes or No.

ECM Fuel Disable Until Ignition Off

Yes

The scan tool displays Yes or No. Yes is displayed if the fuel is disabled during ignition OFF.

ECM Fuel Disable

Inactive

The scan tool displays Active or Inactive. Active is displayed if the fuel disable timed out.

Ignition Voltage Signal

Present

The scan tool displays Present or Not Present. Present is displayed if there is an ignition voltage signal to the TDM.

Mobile Telephone Communications Interface Requested Immobilisation

No

The scan tool displays Active or Inactive. This is the state of the immobiliser system.

Immobilizer Password Learned

Yes

The scan tool displays Yes or No. This is the state of the immobiliser system programming.
