Balancer Cleaning and Inspection

Special Tools

EN-50431 BSM Backlash Measure Adaptor

For equivalent regional tools, refer to Special Tools .

  1. Remove the coating remains on the balance shaft module gear using an air gun and check the balance shaft module if damaged.


  2. Install the EN-50431 BSM backlash measure adapter and dial gauge as shown in the graphic. Check that EN-50431 adapter is horizontal.
  3. Move the balance shaft module gear using a finger and check the backlash specifications (0.046-0.128 mm).
  4. If the balance shaft module is out of specification, try the procedure (1-3) again.
  5. If the balance shaft module gear backlash does not meet the specification, replace with new balance shaft module.
