Captiva
   

Camshaft Position Sensor Replacement - LD9

Removal Procedure

    Warning: Refer to Battery Disconnect Warning in the Preface section.



  1. Disconnect the negative battery cable.
  2. Remove the engine cover.
  3. Disconnect the camshaft position (CMP) sensor connector.


  4. Remove the timing belt front cover. Refer to Timing Belt Rear Cover Replacement : LD9 .
  5. Remove the CMP sensor bolts.
  6. Remove the CMP sensor from the top.

Installation Procedure

    Caution: Refer to Fastener Caution in the Preface section.



  1. Install the CMP sensor with the bolts.

    2. Tighten
    Tighten the camshaft position sensor bolts to 8 N·m (71 lb in).



  2. Install the timing belt front cover. Refer to Timing Belt Rear Cover Replacement : LD9 .
  3. Connect the CMP sensor connector.
  4. Install the engine cover.
  5. Connect the negative battery cable.
