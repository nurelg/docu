Circuit/System Description

The driver side window consists of a driver window switch and a window motor capable of express up and express down functions. When the driver window switch is pressed or pulled, the voltage within the appropriate signal circuit drops and the window motor will then perform the commanded function.

Diagnostic Aids

A window motor that has not been normalised will no longer perform the express up and express down functions, this may occur during the following conditions:

    • When a window motor has been disconnected
    • When a door harness has been disconnected
    • When the battery has been disconnected or replaced

To normalize the window motor refer to Power Window Reinitialisation .

Circuit/System Verification

Command the M74D window motor--driver UP, DOWN, express UP and express DOWN by using the S79D window switch-driver. The M74D window motor--driver should perform the UP, DOWN, express UP and express DOWN commands.

Circuit/System Testing
  1. Ignition OFF, disconnect the harness connector at the S79D window switch-driver.
  2. Test for less than 10 Ω between the ground circuit terminal listed below and ground.
    3. • Ground terminal P (left hand drive)
    • Ground terminal L (right hand drive)
    ⇒If greater than the specified range, test the ground circuit for an open/high resistance.
  3. Ignition ON, verify that a test lamp illuminates between the ignition circuit terminal listed below and ground.
    4. • Ignition terminal K (left hand drive)
    • Ignition terminal R (right hand drive)
    ⇒If the test lamp does not illuminate, test the ignition circuit for a short to ground or an open/high resistance.
  4. Ignition OFF, connect the harness connector at the S79D window switch-driver and disconnect the harness connector at the M74D window motor-driver.
  5. Test for less than 10 Ω between the ground circuit terminal 3 and ground.
    6. ⇒If greater than the specified range, test the ground circuit for an open/high resistance.
  6. Ignition ON, verify that a test lamp illuminates between the ignition circuit terminal 6 and ground.
    7. ⇒If the test lamp does not illuminate, test the ignition circuit for a short to ground or an open/high resistance.
  7. Connect a test lamp between the control circuit terminal 5 and B+.
  8. Command the driver window UP by using the appropriate switch on the S79D window switch-driver, the test lamp should illuminate during the UP command.
    9. ⇒If the test lamp is always OFF, test the control circuit for a short to voltage or an open/high resistance. If the circuit tests normal, test or replace the S79D window switch-driver.
    ⇒If the test lamp is always ON, test the control circuit for a short to ground. If the circuit tests normal, test or replace the S79D window switch-driver
  9. Connect a test lamp between the control circuit terminal 2 and B+.
  10. Command the driver window DOWN by using the switch on the S79D window switch-driver, the test lamp should illuminate during the DOWN command.
    11. ⇒If the test lamp is always OFF, test the control circuit for a short to voltage or an open/high resistance. If the circuit tests normal, test or replace the S79D window switch-driver.
    ⇒If the test lamp is always ON, test the control circuit for a short to ground. If the circuit tests normal, test or replace the S79D window switch-driver.
  11. Connect a test lamp between the control circuit terminal 1 and ground.
  12. Command the driver window express function by pressing the switch past the first detent and releasing, the test lamp should illuminate during the express function.
    13. ⇒If the test lamp is always OFF, test the control circuit for a short to ground or an open/high resistance. If the circuit tests normal, test or replace the S79D window switch-driver.
    ⇒If the test lamp is always ON, test the control circuit for a short to voltage. If the circuit tests normal, test or replace the S79D window switch-driver.
  13. If all circuits test normal, replace the M74D window motor-driver

Component Testing

Driver Window Switch

  1. Ignition OFF, disconnect the harness connector at the S79D window switch-driver.
  2. Test for infinite resistance between the terminals listed below:
    3. • Terminal L and terminal K (left hand drive)
    • Terminal P and terminal R (right hand drive)
    • Terminal M and terminal P (left hand drive)
    • Terminal N and terminal L (right hand drive)
    • Terminal J and terminal P (left hand drive)
    • Terminal S and terminal L (right hand drive)
    ⇒If less than the specified value, replace the S79D window switch-driver.
  3. Test for less than 2 Ω between the terminals listed below with the switch in the UP position.
    4. • Terminal M and terminal P (left hand drive)
    • Terminal N and terminal L (right hand drive)
    ⇒If greater than the specified range, replace the Driver Window switch.
  4. Test for less than 2 Ω between the terminals listed below with the switch in the DOWN position.
    5. • Terminal J and terminal P (left hand drive)
    • Terminal S and terminal L (right hand drive)
    ⇒If greater than the specified value, replace the S79D window switch-driver.
  5. Test for less than 2 ohms between the terminals listed below with the switch in the EXPRESS position.
    6. • Terminal L and terminal K (left hand drive)
    • Terminal P and terminal R (right hand drive)
    ⇒If greater than the specified value, replace the S79D window switch-driver.
