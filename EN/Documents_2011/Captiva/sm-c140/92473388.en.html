Captiva
   

Cylinder Head Assemble



    Note: 

       • If the valves have been used in a running engine, you must clean the valve seats before reassembly.
       • Coat the valve shafts with NEW engine oil.
       • Take extreme care to prevent damage to the valve guides.

  1. Install the valves (5).


  2. Install NEW valve stem oil seals using the EN-46116 installer (1).


  3. Install the valve springs (3).
  4. Install the valve spring retainers (2).


    5. Warning: Compressed valve springs have high tension against the valve spring compressor. Valve springs that are not properly compressed by or released from the valve spring compressor can be ejected from the valve spring compressor with intense force. Use care when compressing or releasing the valve spring with the valve spring compressor and when removing or installing the valve stem keys. Failing to use care may cause personal injury.

    Note: Take extreme care to prevent damage to the valve guides.

  5. Install the EN-48246 compressor (1) with EN-48247 adapter (2).


  6. Install the 32 valve stem keys (1).
