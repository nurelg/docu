Body Control Module Scan Tool Output Controls

Output Control

Description

Accessory/Retained Accessory Power Relay

The BCM actuates the retained accessory power relay when you select On. The retained accessory power relay should turn on allowing specific vehicle functions to operate.

All Doors Lock

The BCM activates all doors lock when you select Lock. All doors should lock.

Reverse Lamps

The BCM actuates the reverse lamps relay when you select On. The reverse lamps should illuminate until commanded Off. Engine must be running.

Brake Pedal Position Sensor Learn

The BCM activates the brake pedal position sensor learn when you select Learn. The brake pedal position sensor learn procedure should start.

Courtesy Lamps

The BCM activates the courtesy lamps when you select On. The value should be 100%. The courtesy lamps should illuminate until commanded Off.

Daytime Running Lamps

The BCM activates the daytime running lamps when you select On. The daytime running lamps should illuminate until commanded Off.

Driver Door Unlock

The BCM pulses the driver door unlock motor when you select Unlock. The driver door should unlock.

Front Fog Lamps

The BCM actuates the front fog lamps relay when you select On. The front fog lamps relay should turn on.

High Beams

The BCM activates the high beam command when you select On. The high beam lights should illuminate. The headlamps have to be on.

Horn Relay

The BCM activates the horn when you select On. The horn should sound.

Incandescent Dimming

The BCM dims the incandescent dimming from 0-100 percent when you select INCREASE or DECREASE by 20 percent increments.

LED Backlight Dimming

The BCM dims the LED backlight dimming from 0-100 percent when you select INCREASE or DECREASE by 20 percent increments.

Left Front Indicator Lamp

The BCM activates the left front indicator lamp when you select On. The left front indicator lamp should illuminate until commanded Off.

Left Rear Turn Signal Lamp

The BCM activates the left rear indicator lamp when you select On. The left rear indicator lamp should illuminate until commanded Off.

Liftgate Window Unlatch

This output control is used to command the liftglass switch On and Off.

Dipped Beam Lamps

The BCM activates the dipped beams when you select On. The dipped beam lights should turn on.

Park Lamps

The BCM activates the parking lamps when you select On. The parking lamps should illuminate.

Passenger Door(s) Unlock

The BCM pulses the passenger door unlock motor when you select Unlock. The passenger door should unlock.

Rear Body Closure Unlatch

The BCM actuates the rear compartment lid release actuator when you select Unlatch. The trunk or liftgate should open.

Rear Demister

This output control is used to command the rear defogger On and Off.

Rear Fog Lamps

This output control is used to command the rear fog lamps On and Off.

Right Front Indicator Lamp

The BCM activates the right front indicator lamp when you select On. The right front indicator lamp should illuminate.

Right Rear Turn Signal Lamp

The BCM activates the right rear turn signal lamp when you select On. The right rear turn signal lamp should illuminate until commanded Off.

Run/Crank Relay

The BCM actuates the run/crank relay when you select On. The run/crank relay should turn on.

Setup SDM Part Number in BCM

This output control is used to set up the SDM primary key into the BCM.

Shift Lock Solenoid Actuator

This output control is used to command the shift lock solenoid On and Off.

Windscreen Washer Motor

The BCM actuates the windshield washer motor when you select On. The windshield washer should turn on.

Windshield Wiper High Speed Relay

The BCM actuates the windscreen wiper high speed relay when you select On. The windscreen wiper high speed relay should turn on.

Windshield Wiper Motor Relay

The BCM actuates the windscreen wiper motor relay when you select On. The windscreen wiper motor relay should turn on.
