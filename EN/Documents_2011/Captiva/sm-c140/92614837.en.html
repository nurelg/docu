Clutch Actuator Cylinder Replacement

Removal Procedure
  1. Remove the transmission. Refer to Transmission Replacement : 2.4 Engine  → Diesel Engine .


  2. Loosen the clutch actuator cylinder pipe nut (1) from the clutch actuator cylinder.


  3. Remove 3 clutch actuator cylinder bolts (3).
  4. Remove the clutch actuator cylinder (1).

    5. Note: Discard the clutch actuator cylinder seal.

  5. Remove the clutch actuator cylinder seal (2).

Installation Procedure
  1. Clean the 3 clutch actuator cylinder threaded holes in the transmission housing.


  2. Install a NEW clutch actuator cylinder seal (2).
  3. Install the clutch actuator cylinder (1).

    4. Caution: Refer to Fastener Caution in the Preface section.

  4. Install the 3 NEW clutch actuator cylinder bolts (3) and tighten to 10 Nm (89 lb in).


  5. Install the clutch actuator cylinder pipe nut (1) and tighten to 18 Nm (13 lb ft).
  6. Install the transmission. Refer to Transmission Replacement : 2.4 Engine  → Diesel Engine .
