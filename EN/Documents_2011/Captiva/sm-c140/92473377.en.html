Engine Flywheel Installation



  1. Install the engine flywheel (2).

    2. Caution: Refer to Fastener Caution in the Preface section.

    Note: Clean thread before inserting the bolt with locking compound. The maximum installation time including torque check is 10 minutes.

  2. Install the engine flywheel bolts (1).

    3. Tighten

        • Engine Flywheel Bolt (M/T): 160 N·m (118 lb ft)
        • Engine Flexible Plate Bolt (A/T): 60 N·m + 45° ± 4° (44.2 lb ft + 45° ± 4°)
