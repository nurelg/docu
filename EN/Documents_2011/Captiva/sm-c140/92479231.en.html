Captiva
   

Turbocharger Installation



  1. Install the turbocharger (3) and new gasket (4).

    2. Caution: Refer to Fastener Caution in the Preface section.

  2. Install the turbocharger lower retaining nuts (2) and tighten to 30 N·m (22.1 lb ft).
  3. Install the turbocharger upper retaining nuts (1) and tighten to 30 N·m (22.1 lb ft).


  4. Connect the turbocharger vacuum line.
  5. Install the exhaust temperature sensor 1 and tighten to 45 N·m (33.2 lb ft). Refer to Exhaust Temperature Sensor Replacement - Position 1 .
  6. Install the turbocharger oil return pipe. Refer to Turbocharger Oil Return Pipe Installation .
  7. Install the turbocharger oil feed pipe. Refer to Turbocharger Oil Feed Pipe Installation .
  8. Install the turbocharger coolant feed pipe. Refer to Turbocharger Coolant Feed Pipe Installation .
  9. Install the turbocharger coolant return pipe. Refer to Turbocharger Coolant Return Pipe Installation .
  10. Install the DPF. Refer to Exhaust Particulate Filter Installation .
  11. Install the turbocharger heat shield. Refer to Turbocharger Heat Shield Installation .
