Valve Stem Oil Seal and Valve Spring Replacement - LE5 or LE9

Special Tools

    • EN-36017 Valve Guide Seal Remover
    • EN-43649 Valve Spring Compressor
    • EN-43649-10 Valve Spring Compressor Adapter Set
    • EN-43653 Flywheel Holding Tool

For equivalent regional tools refer to Special Tools : LE5 or LE9 .

Removal Procedure

Note: Prevent the crankshaft from rotating clockwise or anticlockwise before using compressed air in the cylinder. Rotation of the crankshaft may cause damage to EN-43653 holding tool .

  1. If equipped, with manual gearbox leave the transmission in first gear with the vehicle on the earth and the parking brake set.
  2. If equipped, with automatic transmission, remove the starter. Refer to Starter Replacement : LNQ  → LF1  → LE5 .
  3. Install EN-43653 holding tool .
  4. Remove the camshaft. Refer to Intake Camshaft and Valve Lifter Replacement : LE5 or LE9 or Exhaust Camshaft and Valve Lifter Replacement : LE5 or LE9 .


  5. Remove the camshaft roller followers (1).


  6. Install the EN-43649-10 adapter set (1) to the front and rear of the cylinder head.
  7. Install the cross bars and locks (2) of the EN-43649 compressor to the valve spring compressor adapters.
  8. Remove the spark plugs. Refer to Spark Plug Replacement .
  9. Install an air hose adapter into the spark plug hole.
  10. Attach an air hose to the adapter and pressurize the cylinder to 690 kPa (100 psi).
  11. Compress the valve spring using the valve spring compressor (3).


  12. Remove the valve spring keepers (1).
  13. Remove the valve spring (3) and retainer (2).


  14. Use the EN-36017 seal remover (1) to remove the valve seal.

Installation Procedure


  1. Using the EN-36017 seal remover (1) install the NEW valve seal. Fully seat the seal onto the valve guide.


  2. Install the valve spring (3) and retainer (2).
  3. Compress the valve spring using the valve spring compressor.
  4. Install the valve spring keepers (1).
  5. Disconnect the air hose and air hose adapter.


  6. Remove the EN-43649 compressor and EN-43649-10 adapter set from the cylinder head.
  7. Install the spark plugs. Refer to Spark Plug Replacement .


  8. Install the camshaft roller followers.
  9. Install the camshaft. Refer to Intake Camshaft and Valve Lifter Replacement : LE5 or LE9 or Exhaust Camshaft and Valve Lifter Replacement : LE5 or LE9 .
  10. If used, remove the EN-43653 holding tool .
  11. If removed, install the starter. Refer to Starter Replacement : LNQ  → LF1  → LE5 .
