Wiring Harness Installation



  1. Install a NEW automatic transmission wiring connector seal (46) in the bore for the connector. The taper side of the seal goes up.
  2. Install a NEW TCM seal (72) on the connector.
  3. Install the transmission wiring harness assembly (33) into the transmission case assembly through the wiring harness bore.


  4. Align the tab of the connector in the transmission case.
  5. Install the wiring harness connector retainer (69).


  6. Apply automatic transmission fluid to the new transmission fluid temperature sensor O-ring seal (32).
  7. Install the new transmission fluid temperature sensor O-ring seal (32) into the groove of the transmission fluid temperature sensor (36).
  8. Install the transmission fluid temperature sensor (36) into the front control valve body fluid temperature sensor bore.
  9. Install the transmission fluid temperature sensor clip (49).

    10. Caution: Refer to Fastener Caution in the Preface section.

  10. Install the 1 - M6 x 1.0 x 55 mm fluid temperature sensor bolt (31) and tighten to 10 N·m (89 lb in).


  11. Connect the 8 solenoid wire connectors as follows:
    12. • 1 - blue
    • 2 - black
    • 3 - green
    • 4 - black
    • 5 - blue
    • 6 - grey
    • 7 - brown
    • 8 - black
  12. Position the wiring harness in the wiring harness clip.
