Captiva
   

Heater Core Replacement

Removal Procedure


    Note: The left-hand drive is shown. The right-hand drive is similar.

  1. Remove the instrument panel from the vehicle. Refer to Instrument Panel Assembly Replacement : Long Body  → Short Body .
  2. Remove the A/C module from the vehicle. Refer to Air Conditioning and Heater Module Assembly Removal and Installation .
  3. Disconnect the heater hoses from the heater core.
  4. Remove the thermistor wiring harness connector.


  5. Remove the mode motor ventilation link.


  6. Remove the heater core side cover with the screws.


  7. Remove the heater core.

Installation Procedure

    Warning: Refer to Battery Disconnect Warning in the Preface section.



  1. Install the heater core into the case.


  2. Install the heater core side cover.
  3. Install and tighten the screws that secure the heater core cover to the air conditioning module.


  4. Install the mode door ventilation link and screw.


  5. Connect the thermistor connector.
  6. Connect the heater hoses to the heater core.
  7. Install the A/C module. Refer to Air Conditioning and Heater Module Assembly Removal and Installation .
  8. Install the instrument panel. Refer to Instrument Panel Assembly Replacement : Long Body  → Short Body .
