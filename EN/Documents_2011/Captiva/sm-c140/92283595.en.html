1. Fuel Door Latch Inoperative

Circuit/System Description

The rear fuse block, upon receipt of a lock switch lock or unlock signal, will switch the fuel filler door release actuator control circuits to battery positive voltage. The opposing side of the fuel filler door release actuator is connected to ground provided by the rear fuse block through the other door lock actuator control circuit and will lock or unlock as commanded.

Circuit/System Verification

Observe the function of the door latches while commanding the doors to LOCK and UNLOCK by toggling the driver door lock switch between the LOCK and UNLOCK. All doors and the fuel door release actuator should perform the LOCK and UNLOCK functions while changing between the commanded states.

Circuit/System Testing
  1. Ignition OFF, disconnect the wiring harness plug at the fuel door latch actuator.
  2. Connect a test lamp between control circuit terminal 1 and control circuit terminal 2.
  3. Ignition ON, command the door locks to LOCK and UNLOCK by toggling the driver door lock switch between the LOCK and UNLOCK. The test lamp should illuminate while commanding the LOCK and UNLOCK states.
    4. ⇒If the test lamp is always ON, test either control circuit for a short to voltage.
    ⇒If the test lamp remains OFF, test either control circuit for a short to ground or open/high resistance.
  4. If all circuits test normal, test or replace the fuel door latch actuator.

Component Testing
  1. Ignition OFF, disconnect the wiring harness plug at the fuel door latch actuator.
  2. Install a 15A fused jumper wire between one of the control terminals and 12V. Momentarily install a jumper wire between the other control terminal and ground. Reverse the jumper wires at least two times, the fuel door latch actuator should perform the LOCK/UNLOCK functions.
    3. ⇒If it does not perform as specified, replace the fuel door latch actuator.
