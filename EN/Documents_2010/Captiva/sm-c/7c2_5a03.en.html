Diagnostic Trouble Code (DTC) P0562

System Voltage Low

Conditions for Running the DTC

  • Emergency mode is OFF.
  • No failure condition of BUS OFF/ECM No ID.
  • No failure of engine speed.
  • Engine speed is more than 400 RPM.

Conditions for Setting the DTC

  • The system voltage is less than 9 volts for 20 seconds.

Action Taken When the DTC Sets

  • TCM will request the illumination of MIL.
  • TCM records the operating condition when the Conditions for Setting the DTC are met. TCM stores this information as Failure Record.
  • TCM stores DTC in TCM history.
  • TCM turns Emergency mode ON.
When the Emergency mode is ON, the following conditions occur:
    The transmission will operate in 4th gear when the transmission is shifted into D ,M+ or 4 th range.
    The transmission will operate in 2th gear when the transmission is shifted into 2 or M- range.
    Learn control is disabled.
    Lock up control is disabled.
    Adaptive control is disabled.
    N-D shifting is disabled.
    ABS function is disabled.

Conditions for Clearing the MIL/DTC

  • The MIL turns OFF when no failure is detected for three consecutive driving cycles.
  • The scan tool can clear the DTC from the TCM history.
  • TCM clears the DTC from the TCM history memory after forty consecutive warm-up cycles without fault.

Cause of Failure

  • TCM wiring harness and connectors
  • Battery
  • TCM

Diagnostic Trouble Code (DTC) P0601

Transmission Control Module (TCM) Read Only Memory (ROM)

Conditions for Running the DTC

  • TCM runs the program to detect internal fault when the engine is running.

Conditions for Setting the DTC

  • TCM detects the calculated checksum is different from the correct checksum value in ROM.

Action Taken When the DTC Sets

  • TCM will request the illumination of MIL.
  • TCM records the operating condition when the Conditions for Setting the DTC are met. TCM stores this information as Failure Record.
  • TCM stores DTC in TCM history.
  • TCM turns Emergency mode ON.
When the Emergency mode is ON, the following conditions occur:
    The transmission will operate in 4 th gear when the transmission is shifted into D , M+, or 4th gear range
    The transmission will operate in 2th gear when the transmission is shifted into 2 or M- range.
    Learn control is disabled.
    Lock up control is disabled.
    Adaptive control is disabled.
    N-D shifting is disabled.
    ABS function is disabled.

Conditions for Clearing the MIL/DTC

  • The MIL turns OFF when no failure is detected for three consecutive driving cycles.
  • The scan tool can clear the DTC from the TCM history.
  • TCM clears the DTC from the TCM history memory after forty consecutive warm-up cycles without fault.

Cause of Failure

  • Harness or connector between TCM and ECM
  • ECM
  • TCM

    The transmission will operate in 4 th gear when the transmission is shifted into D , M+, or 4th gear range

    Learn control is disabled.
    Lock up control is disabled.
    Adaptive control is disabled.
    N-D shifting is disabled.
    ABS function is disabled.

Conditions for Clearing the SVS Lamp/DTC

  • TCM turns OFF the SVS lamp when no failure is detected for three consecutive driving cycles.
  • The scan tool can clear the DTC from the TCM history.
  • TCM clears the DTC from the TCM history memory after forty consecutive warm-up cycles without fault.

Cause of Failure

  • Harness or connector between TCM and ECM
  • ECM
  • TCM

Diagnostic Trouble Code (DTC) P0711

Conditions for Running the DTC

  • No failure condition of gear range.
  • Emergency mode is OFF.
  • The TFT at startup is less than 20°C (68°F).

Conditions for Setting the DTC

  • The TFT has not changed by more than 5°C (9°F) since startup for 10 minutes.

Action Taken When the DTC Sets

  • TCM will request the illumination of MIL.
  • TCM records the operating condition when the Conditions for Setting the DTC are met.TCM stores this information as Failure Record.
  • TCM stores DTC in TCM history.
  • TCM uses 30°C (86°F) as a default transmission oil temperature and will start the self learning control after setting TFT as 111°C (232°F) 15 minutes later.

  • Uphill and downhill control are not activated until the TFT is greater than 40°C (104°F) for 5 seconds.

  • The MIL turns OFF when no failure is detected for three consecutive driving cycles.
  • The scan tool can clear the DTC from the TCM history.

  • Wiring harness or connector between Transmission Fluid Temperature (TFT) sensor and TCM

Input Speed Sensor Circuit Low Voltage

Conditions for Running the DTC

Fail Case 1
  • If the vehicle speed is 66km/h(41mph) or transmission oil temperature is greater than 20°C (68°F), the gear range is changed from P,R,N range to others for 2.5 seconds or greater.
  • If the transmission oil temperature is 0-20°C (32-68°F), the gear range is changed from P,R,N range to others for 10 seconds or greater.
  • If the transmission oil temperature is less than 0°C (32°F), the gear range is changed from P,R,N range to others for 180 seconds or greater.
  • Output speed is greater than 600 RPM.
  • No gear selector failure.
  • The transmission is not shifting.
  • TCM is not in ND control.
  • The commanded gear is 2 or greater.
  • No TFT DTCs.
  • Emergency mode is OFF.
Fail Case 2
  • System voltage is 10.2-15.5 volts.
  • Emergency mode is OFF
Fail Case 3 (For the vehicle equipped with ABS)
  • If the vehicle speed is 66km/h(41mph) or transmission oil temperature is greater than 20°C (68°F), the gear range is changed from P,R,N range to others for 2.5 seconds or greater.
  • If the transmission oil temperature is 0-20°C (32-68°F), the gear range is changed from P,R,N range to others for 10 seconds or greater.
  • If the transmission oil temperature is less than 0°C (32°F), the gear range is changed from P,R,N range to others for 180 seconds or greater.
  • Output speed is greater than 600 RPM.
  • No gear selector failure.
  • The transmission is not shifting.
  • TCM is not in ND control.
  • The commanded gear is 2 or greater.
  • No TFT DTCs.
  • Emergency mode is OFF.

Conditions for Setting the DTC

Fail Case 1
  • TCM does not detect any pulses from the input speed sensor while detecting 3,000 pulses of the output speed sensor signal.
Fail Case 2
  • Input speed sensor signal circuit is open or shorted for 30 seconds or greater.
Fail Case 3 (For the vehicle equipped with ABS)
  • Input speed RPM is equal to 0 and ABS vehicle speed is greater than 20 km/h (12 mph) for 30 seconds or greater.

Action Taken When the DTC Sets

  • TCM will request the illumination of MIL.
  • TCM records the operating condition when the Conditions for Setting the DTC are met.TCM stores this information as Failure Record.

  • TCM turns Emergency mode ON.

    The transmission will operate in 4 th gear when the transmission is shifted into D , M+, or 4 th gear range.
    The transmission will operate in 2th gear when the transmission is shifted into 2 or M- range.
    Learn control is disabled.
    Lock up control is disabled.
    Adaptive control is disabled.
    N-D shifting is disabled.
    ABS function is disabled.

Conditions for Clearing the MIL/DTC

  • The MIL turns OFF when no failure is detected for three consecutive driving cycles.
  • The scan tool can clear the DTC from the TCM history.
  • TCM clears the DTC from the TCM history memory after forty consecutive warm-up cycles without fault.

Cause of Failure

  • Wiring harness or connector between Input Speed Sensor and TCM
  • Input Speed Sensor
