Output Shaft Assembly

Tools Required

DT-48240 Output 5th/Reverse Synchronzer Installer
DT-48239 Output 5th Bearing Collar Installer
DT-48237 Output 3rd/4th Gear Installer

Assembly Procedure

    Important : Lubricate all components in transaxle fluid before installation.

  1. Install the thrust washer using the DT-48240 and a hydraulic press.
  2. Install the collar using the DT-48240 and a hydraulic press.



  3. Install the needle bearings (3).
  4. Install the reverse gear (2).
  5. Install the reverse gear blocking ring (1).



  6. Install the 5th/Reverse synchronizer using the DT-48240 and a hydraulic press.
    7. Important
    • The raised center hub of the 5th/Reverse synchronizer assembly faces the 5th gear.
    • Line up the blocking ring tabs and the hub keyways while pressing.



  7. Install the bearing collar using the DT-48239 and a hydraulic press.



  8. Install the 5th gear blocking ring (3).
  9. Install the two needle bearings (2).
  10. Install the 5th gear (1).



  11. Install the 3rd gear using the DT-48237 and a hydraulic press.

    12. Important : The 3rd gear will install very tight. Ensure that the mating surfaces are clean.




  12. Apply Sealer-Retaining (Permatex 620) to the circumference of the input shaft before installing the 4th gear.
  13. Install the 4th gear using the DT-48237 and a hydraulic press.

    14. Important : The 4th gear will install very tight. Ensure that the mating surfaces are clean.




Input Shaft Assembly

Tools Required

DT-48238 Input 5th Gear Installer
DT-48236 Input 3rd/4th Synchronizer Installer

Assembly Procedure

    Important
    • Lubricate all components with transaxle fluid before installation.
    • The machined surface faces the needle thrust bearing.

  1. Apply Sealer-Retaining (Permatex 620) to the circumference of the input shaft before installing the 5th gear.
  2. Install the 5th gear using the DT-48238 and a hydraulic press.



  3. Install the thrust bearing (2).
  4. Install the two needle bearings (1).



  5. Install the 3rd gear (4).
  6. Install the 3rd gear blocking ring (3).
  7. Install the thrust bearing (2).
  8. Install the thrust washer (1).



  9. Install the 3rd/4th synchronizer using the DT-48236 and a hydraulic press.
    10. Important
    • The machined surface of the synchronizer faces the thrust washer.
    • Make sure to align the slots in the synchronizer with the gear blocking ring tabs.



  10. Install the bearing collar using the DT-48236 and a hydraulic press.



  11. Install the 4th gear blocking ring (4).
  12. Install the needle bearing (3).
  13. Install the 4th gear (2).
  14. Install the thrust washer (1).



  15. Install the sleeve bearing using an appropriate tool.



Interm Shaft Assembly

Tools Required

DT-48235 Interm 1st/2nd synchronizer installer

Assembly Procedure

    Important : Lubricate all components with transaxle fluid before installation.

  1. Install the needle bearing (5).
  2. Install the 2nd gear (4).
  3. Install the 2nd gear inner cone (3).
  4. Install the 2nd gear blocking ring (2).
  5. Install the 2nd gear outer cone (1).



  6. Install the 1st/2nd synchronizer using the DT-48235 and a hydraulic press.
    7. Important
    • Make sure to properly align the synchronizer to the gear and notches in the blocking ring.
    • Observe the orientation of the synchronizer.The shoulder of the synchronizer faces 1st gear.



  7. Install the thrust washer (2) and the bearing collar (1) using the DT-48235 and a hydraulic press.



  8. Install the 1st gear outer cone (3).
  9. Install the 1st gear blocking ring (2).
  10. Install the 1st gear inner cone (1).



    11. Important : The steps in both roller bearings must fall into the 1st gear.

  11. Install the roller bearing.
  12. Install the 1st gear.
  13. Install the roller bearing.



  14. Install the thrust washer using the DT-48235 and a hydraulic press.



Shift Shaft Assembly

Assembly Procedure

  1. Install the angle disc (4) onto the shift rod (5).
  2. Install the 1st/2nd pressure spring (3).
  3. Install the inner spring seat (2).
  4. Install new snap rings (1).



  5. Install the shift cover (4).
  6. Install the vent cap (3).
  7. Install the shift lever (1).
  8. Install a new coiled pin (2).



  9. Install the 5th/reverse pressure spring (1).
  10. Install the outer spring seat (2).
  11. Install the shift interlock (3) into the shift finger (4).

    12. Important : Properly support the shift finger (4) when installing the coiled pin. Do not support the shift finger (4) with the shift interlock (3).

  12. Install a new coiled pin (5).
  13. Install a new coiled pin (6).



Major Component Assembly

Tools Required

DT-48179 Universal Driver Handle
DT-48182 Needle Sleeve Remover/Installer
DT-48225 Input Shaft Holder
DT-48226 Transaxle Housing Groove Ball Bearing Remover/Installer
DT-48227 Roller Bearing Remover/Installer
DT-48228 Clutch Housing Groove Ball Bearing Remover/Installer
DT-48230 Differential Bearing Race and Shaft Seal Ring Installer
DT-48231 Input and Outupt Shaft Pusher/Puller
DT-48232 Input Shaft Synchro Body Protector

Assembly Procedure

  1. Install a new shaft seal ring to the clutch housing using the DT-48179 and the DT-48230.



  2. Install the Interm shaft groove ball bearing to the clutch housing using the DT-48179 and the DT-48228.



  3. Install the snap ring to the interm shaft groove ball bearing.



  4. Install the input shaft roller bearing to the clutch housing using the DT-48179 and the DT-48227.



  5. Install a new shaft seal ring to the transaxle housing using the DT-48179 and DT-48230.



  6. Install the input shaft and output shaft groove ball bearings into the transaxle housing using the DT-48179 and the DT-48226.



  7. Install the needle sleeve flush with the casting in the transaxle housing using the DT-48179 and the DT-48182.



  8. Install the reverse shifting blocker(1), the torsion spring(2), and a new thread pin(3) to the transaxle housing

    9. Tighten
    Tighten the thread pin to 6~9 N•m (53~80 lb-in).




  9. Install the output shaft roller bearing to the clutch housing.

    10. Important : Be careful the direction of roller bearing.




  10. Install Interm (1), input (2) and output (3) shaft assembly with the respective shift forks in the clutch housing.
    • Install the DT-48232 and the input shaft retainer plug.
    • Install the Interm (1) and input (2) shaft assembly with the respective shift forks.

    Important : Position the Interm shaft to the groove ball bearing side.

    • Install the output shaft assembly and the shift fork.
    • After checking the gears engagement, install each the shaft assembly completely.
    • Remove the DT-48232 and the input shaft retainer plug.



  11. Install the transaxle housing on the clutch housing.
    1. Input and output shaft must engage in the groove ball bearings in the transaxle housing.
    2. The Interm shaft and the shift forks must sit in their opposing bearings in the transaxle housing.
    3. Install the DT-48231 on the transaxle housing.
    • Tighten the four bolts (1).
    • Screw the spindles into the input and output shaft.
    1. Draw the input and output shaft evenly into the groove ball bearings in the transaxle housing.

    • Hold the spindles (2) using a suitable tool.
    • Turn via nuts (3) in direction of arrow.
    1. Remove the DT-48231 from the transaxle housing.
    • Unscrew the spindles into the input and output shaft.
    • Remove the four bolts(1).

    Important : If large forces are necessary when actuating the DT-48231,separate the transaxle housing and the clutch housing again and check positioning of the shaft and shift forks. Repeat operation.




  12. Shift the transaxle into 4th and 5th gear using a screwdriver in an upward direction.
  13. Install new input and output shaft retainer plugs while holding the transaxle into 4th and 5th gear in an upward direction.

    14. Tighten
    Tighten the input shaft retainer plug to 120~130 N•m (89~96 lb-ft).
    Tighten the output shaft retainer plug to 120~130 N•m (89~96 lb-ft).




  14. Apply sealant (Loctite 5900) to the transaxle housing rear cover.

    15. Important : Leave area free in order not to block up the oil pipe.




  15. Install the oil pipe and transaxle housing rear cover.

    16. Important : The oil pipe only installs in one direction. Ensure the oil pipe is completely seated in the gear shafts.




  16. Install the transaxle housing rear cover bolts.

    17. Tighten
    Tighten the rear cover bolts to 20~25 N•m (15~18 lb-ft).




  17. Remove the clutch housing from the transaxle housing.

    18. Important : In line with the Interm shaft, use suitable tools to assist in the housing separation.




  18. Install the idle shift rail (1) into the transaxle housing.
  19. Install the idle shift rail bushing (2) to the transaxle housing.
  20. Install the drain channel (3) into the transaxle housing.



  21. Install the backup lamp switch and the washer to the transaxle housing.

    22. Tighten
    Tighten the backup lamp switch to 20 N•m (15 lb-ft).




  22. Install the shift shaft assembly.
    • Apply sealer (Loctite 5203) to the transaxle housing to the gear shift cover mating surface.
    • Position the transaxle into neutral.
    • Install the shift shaft assembly (3).
    • Install the select control lever (2).
    • Install the shift retaining bolts.

    Tighten
    Tighten the shift retaining bolts to 20~25 N•m (15~18l b-ft).

    • Install the shift guide bolt (1).

    Tighten
    Tighten the shift guide bolt to 20~25 N•m (15~18 lb-ft).




  23. Install the differential bearing race and the shim to the transaxle housing using the DT-48179 and the DT-48230. Refer to "Shimming Procedures" in this section.



  24. Install the differential bearing race and the shim to the clutch housing using the DT-48230 (For FWD) or the DT-48533 (For AWD) with the DT-48179. Refer to "Shimming Procedures" in this section.



  25. Install the magnet into the transaxle housing.



  26. Install the differential into the transaxle housing.



  27. Apply sealer (Loctite 5203) to the transaxle housing to clutch housing mating surface.
  28. Install the clutch housing to the transaxle housing.
    29. Important
    • The input and output shaft must engage in its bearing in the clutch housing.
    • The Gear shift forks sit in their opposing bearings in the clutch housing.



  29. Hold the input shaft with the DT-48225.
  30. Draw the Interm shaft into the Interm shaft groove ball bearing with old retainer plug.
    • Shift the transaxle into any gear.



  31. Install the clutch housing to transaxle housing bolts.

    32. Tighten
    Tighten the clutch housing to transaxle housing bolts to 20~25 N•m (15~18 lb-ft).

  32. Remove the old retainer plug from the Interm shaft, and Install a new retainer plug.

    33. Tighten
    Tighten the interm shaft retainer plug to 90~100 N•m (66~74 lb-ft).

  33. Remove the DT-48225.



  34. Install the transaxle housing to clutch housing bolts.

    35. Tighten
    Tighten the transaxle housing to clutch housing bolts to 20~25 N•m (15~18 lb-ft).




  35. Install a new oil pipe into the interm shaft.
  36. Install a new interm shaft sealing plug.



  37. Install the o-ring and concentric slave cylinder.



  38. After installing the pipe screw provisionally, tighten the concentric slave cylinder retaining bolts.

    39. Tighten
    Tighten the concentric slave cylinder retaining bolts to 8~12 N•m (71~106 lb-in).

  39. Tighten the pipe screw.

    40. Tighten
    Tighten the pipe screw to 16 N•m (12 lb-ft).




  40. Install the bushing to the concentric slave cylinder pipe.



  41. Install the shift control cable bracket, the washer, and the bolts.

    42. Tighten
    Tighten the shift control cable bracket bolts to 15~20 N•m (11~15 lb-ft).

  42. Install the transaxle into the vehicle. Refer to "Transaxle Assembly" in this section.



Shimming Procedures

    Tools Required

    DT-48200 transaxle housing fixing
    DT-48241 transaxle housing race plate
    DT-48242 clutch housing race plate
  1. Using a rubber mallet, tap lightly on the transaxle housing mating surface to ensure the bearings are fully seated.



  2. Install the DT-48200 to transaxle housing.
  3. Adjust the zero point setting for each place on the transaxle housing flange surface Using the DT-48200 and the height gage.

    4. Important : Adjust the height of the DT-48200 for the zero point setting.




  4. Measure the three places on the output shaft gear thrust washer surface using the height gage.(average=A)
    • The height from the transaxle housing flange surface to the thrust washer surface.



  5. Place the clutch housing differential bearing race, the differential, the transaxle housing differential bearing race, and the DT-48241 onto the DT-48242 (For FWD) or the DT-48534 (For AWD).



  6. Adjust the zero point setting for each place on the transaxle housing differential bearing race surface using the height gage.



  7. Measure the three places on differential ring gear surface using the height gage.(average=B)
    • The height from the transaxle housing differential bearing race to the differential ring gear surface.



  8. Measure the three places on the DT-48242 (For FWD) or the DT-48534 (For AWD) using the height gage.(average=C)
    • The height from the transaxle housing differential bearing race to clutch housing differential bearing race.



  9. To obtain shim size for transaxle housing shim use the following formula.
    10. S1(For Both FWD and AWD) = 62.92mm - A - B
  10. To obtain shim size for clutch housing shim use the following formula :
    11. S2 (For FWD) = 156.84mm - C - S1
    S2 (For AWD)= 158.09mm - C - S1
    • SHIM SPEC (Tolerance : -0.03)
    - For FWD, shim size of the transaxle housing and the clutch housing.
    - For AWD, shim size of the transaxle housing.
