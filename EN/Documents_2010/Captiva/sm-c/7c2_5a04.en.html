Diagnostic Trouble Code (DTC) P0731

Incorrect 1st Gear Ratio

Conditions for Running the DTC

  • Current gear is 1 st gear.
  • Engine speed is at least 150 RPM greater than the input speed for 0.5 seconds.
  • Engine torque is between 30-200 N•m (22-147 lb-ft).
  • Input torque fluctuation is 25 N•m (18 lb-ft) or less.
  • Input speed is greater than 400 RPM.
  • Output speed is greater than 200 RPM.
  • The elapsed time is 2 seconds after changing to the shift position (range D,M,2).
  • 1 seconds pass after N-D control.
  • System voltage is 10.5 volts or greater.
  • Transmission fluid Temperature is 20°C (68°F) or greater.
  • Engine speed is 400 RPM or greater.
  • 0.5 seconds pass after shifting.
  • No garage shifting.
  • Emergency mode is OFF.
  • No failure detection of output speed sensor, input speed sensor, selector position, shift solenoid, linear solenoid, CAN BUS OFF, engine RPM, engine torque, throttle, transmission fluid temperature sensor.
  • No failure detection of ABS and wheel speed.

Conditions for Setting the DTC

  • | input speed/output speed X 4.602 | is greater than 300 RPM and | input speed/output speed X 2.220 | is less than 100 RPM.

Action Taken When the DTC Sets

  • TCM will request the illumination of MIL.
  • TCM records the operating condition when the Conditions for Setting the DTC are met. TCM stores this information as Failure Record.
  • TCM stores DTC in TCM history.
  • 1 st gear is inhibited.
  • N-D shifting is disabled.

Conditions for Clearing the MIL/DTC

  • The MIL turns OFF when no failure is detected for three consecutive driving cycles.
  • The scan tool can clear the DTC from the TCM history.
  • TCM clears the DTC from the TCM history memory after forty consecutive warm-up cycles without fault.

Cause of Failure

  • Shift Solenoid (SS) 1 (Stuck OFF)
  • Shift Solenoid (SS) 2 (Stuck ON)
  • Line Pressure Control (PC) Solenoid (Stuck OFF)
  • Inside Valve Body
  • TCM

Diagnostic Trouble Code (DTC) P0732

Incorrect 2nd Gear Ratio

Conditions for Running the DTC

  • Throttle is 10% or greater.
  • Input speed is greater than 400 RPM.
  • Output speed is greater than 200 RPM.
  • 0.5 seconds pass after 2nd coast clutch (B1) apply control finished.
  • The elapsed time is 2 seconds after changing to the shift position (range D,M,2).
  • 1 seconds pass after N-D control.
  • 0.5 seconds pass after shifting control.
  • Transmission fluid Temperature is 20°C (68°F) or greater.
  • Engine speed is 400 RPM or greater.
  • System voltage is 10.5 volts or greater.
  • No shifting.
  • No garage shifting.
  • Emergency mode is OFF.
  • No failure detection of output speed sensor, input speed sensor, selector position, shift solenoid, linear solenoid, CAN BUS OFF, engine RPM, engine torque, throttle, transmission fluid temperature sensor.
  • No failure detection of ABS and wheel speed.

Conditions for Setting the DTC

  • Current gear is 2nd gear.
  • | 1 - current 2nd gear ratio / expected 2nd gear ratio | is greater than 20% of the expected 2nd gear ratio for 5 seconds.

Action Taken When the DTC Sets

  • TCM will request the illumination of MIL.
  • TCM records the operating condition when the Conditions for Setting the DTC are met. TCM stores this information as Failure Record.
  • TCM stores DTC in TCM history.
  • TCM requests the ECM to limit engine torque to 150 N•m (111 lb-ft).
  • TCM turns Emergency mode ON.
When the Emergency mode is ON, the following conditions occur :
    The transmission will operate in 4th gear when the transmission is shifted into D , M+, or 4th gear range.
    The transmission will operate in 2th gear when the transmission is shifted into 2 or M- range.
    Learn control is disabled.
    Lock up control is disabled.
    Adaptive control is disabled.
    N-D shifting is disabled.
    ABS function is disabled.

Conditions for Clearing the MIL/DTC

  • The MIL turns OFF when no failure is detected for three consecutive driving cycles.
  • The scan tool can clear the DTC from the TCM history.
  • TCM clears the DTC from the TCM history memory after forty consecutive warm-up cycles without fault.

Cause of Failure

  • Shift Solenoid (SS) 1 (Stuck ON)
  • Shift Solenoid (SS) 2 (Stuck OFF)
  • Shift Solenoid (SS) 3 (Stuck OFF)
  • Shift Solenoid (SS) 4 (Stuck ON)
  • Forward Clutch (C1), 2nd Coast Brake (B1), 2nd Brake (B2), B5 Brake.
  • Inside Valve Body
  • Torque Converter Clutch (TCC) Lock Up Pressure Control (PC) Solenoid (Stuck ON)
  • Line Pressure Control (PC) Solenoid (Stuck OFF)
  • TCM
