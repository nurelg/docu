Diagnostic Trouble Code (DTC) P1715

Incorrect 1st Gear Ratio (SS5 : OFF, TCC Lock up PC Solenoid (SLU) : OFF)

Conditions for Running the DTC

  • Mode selector is up and down mode.
  • Output speed is 500-1260 RPM.
  • The elapsed time is 2 seconds after changing to the shift position (range 2).
  • 0.5 seconds pass after shifting control.
  • Throttle is 0%.
  • Transmission fluid Temperature is 20°C (68°F) or greater.
  • Engine speed is greater than 400 RPM.
  • System voltage is 10.5 volts or greater.
  • No shifting
  • No N-D shifting
  • No failure detection of output speed sensor, input speed sensor, selector position, shift solenoid, lockup slip, linear solenoid, engine RPM, engine torque, throttle, transmission fluid temperature sensor

Conditions for Setting the DTC

  • Current gear is 1st gear.
  • | 1 - current 1st gear ratio / expected 1st gear ratio | is greater than 20% of the expected 1st gear ratio for 12 seconds.

Action Taken When the DTC Sets

  • TCM will request the illumination of MIL.
  • TCM records the operating condition when the Conditions for Setting the DTC are met. TCM stores this information as Failure Record.
  • TCM stores DTC in TCM history.
  • TCM requests the ECM to limit engine torque to 150 N•m (111 lb-ft).
  • TCM turns Emergency mode ON.
When the Emergency mode is ON, the following conditions occur :
    The transmission will operate in 4th gear when the transmission is shifted into D , M+, or 4 th gear range.
    The transmission will operate in 2th gear when the transmission is shifted into 2 or M- range.
    Learn control is disabled.
    Lock up control is disabled.
    Adaptive control is disabled.
    N-D shifting is disabled.
    ABS function is disabled.

Conditions for Clearing the MIL/DTC

  • The MIL turns OFF when no failure is detected for three consecutive driving cycles.
  • The scan tool can clear the DTC from the TCM history.
  • TCM clears the DTC from the TCM history memory after forty consecutive warm-up cycles without fault.

Cause of Failure

  • Shift Solenoid (SS) 1 (Stuck OFF)
  • Shift Solenoid (SS) 2 (Stuck ON)
  • Line Pressure Control (PC) Solenoid (Stuck OFF)
  • Inside Valve Body
  • TCM

Diagnostic Trouble Code (DTC) U0001

Can Bus OFF Counter Overrun

Conditions for Running the DTC

  • Voltage supplied to the module is in the normal operating voltage range.
  • The vehicle power mode requires communication to occur.

Conditions for Setting the DTC

  • A total non communication condition is detected on CAN serial data circuit.

Action Taken When the DTC Sets

  • TCM will request the illumination of MIL.
  • TCM records the operating condition when the Conditions for Setting the DTC are met. TCM stores this information as Failure Record.
  • TCM stores DTC in TCM history.
  • TCM requests the ECM to limit engine torque to 270 N•m (199 lb-ft).
  • TCM turns Emergency mode ON.
When the Emergency mode is ON, the following conditions occur:
    The transmission will operate in 4th gear when the transmission is shifted into D or M+ range.
    The transmission will operate in 2th gear when the transmission is shifted into 2 or M- range.
    Learn control is disabled.
    Lock up control is disabled.
    Adaptive control is disabled.
    N-D shifting is disabled.
    ABS function is disabled.

Conditions for Clearing the MIL/DTC

  • The MIL turns OFF when no failure is detected for three consecutive driving cycles.
  • The scan tool can clear the DTC from the TCM history.
  • TCM clears the DTC from the TCM history memory after forty consecutive warm-up cycles without fault.
  • A current DTC clears when the malfunction is no longer present.

Cause of Failure

  • Wiring harness or connector
  • Modules
